评估模式：暂时屏蔽“可直接交付执行”标准，只看概念探索价值。

图片说明：本提交版文档使用仓库相对路径，便于后续提交到主干；Codex 桌面本地预览请打开同目录下的 `.local-preview.md`，该文件使用绝对路径并已加入 `.gitignore`。

## 本轮屏蔽项

这一版先不因以下问题大幅扣分：

- 图片中有英文标签、伪 UI 字、说明文字。
- 尺寸不是严格 16:9。
- 画面是否已经能直接交给外包或执行团队拆资产。
- 是否像标准 art direction board 模板。

仍然保留的评判项：

- 3 秒内有没有市场卖点。
- 是否跳出上一轮过暗、过哥特的问题。
- 成人擦边是否更明确且不越界。
- 主角、敌人、Boss、玩法瞬间有没有记忆点。
- 卡牌/车票/判定/清屏是否能继续发展成真实玩法。
- 是否值得进入下一轮融合、重生成或 GDD 美术基准。

## 图 1：N2-A 日光装甲特急

![N2-A 日光装甲特急](../originals/nightrail-n2-a-sunlit-armored-express.png)

概念价值分：95.60

为什么排第一：

- 它解决了上一轮最大问题：夜车不再只有黑暗、哥特、葬礼。
- 成年女性列车长有市场吸引力，但仍是职业角色，不像单纯擦边皮肤。
- 车票、验票器、车厢 lane、敌人 wave、轨道攻击都能继续变成真实系统。
- “日光装甲特急”比传统夜车更有差异，但还保留铁路幻想。

下一轮保留：

- 金色日光、装甲车厢、黄铜验票器。
- 女列车长露肩/高开衩/职业制服的擦边尺度。
- 车票卡牌作为战斗面板的表达。

需要修正：

- 名称里仍叫夜车，可以把视觉时间改成“黄昏/破晓/荒原晨光”，避免和题名冲突。
- 断钩清屏记忆点需要从 N2-D 合并进来。

## 图 2：N2-B 霓虹末班地铁

![N2-B 霓虹末班地铁](../originals/nightrail-n2-b-neon-last-subway.png)

概念价值分：95.30

为什么上升：

- 原评估主要扣在文字标签，现在屏蔽交付标准后，它的方向价值非常高。
- 现代地铁、霓虹车票、彩色路线卡、都市怪潮都更年轻、更商业。
- 成人女列车员首屏很强，和传统哥特夜车形成明显差异。
- 彩色票卡天然适合构筑和属性系统。

风险：

- 容易从 `断轨夜车` 跑成泛赛博地铁。
- 传统哥特/灵异铁路玩家可能会觉得 IP 识别变弱。

下一轮用途：

- 不替代 N2-A 主线。
- 用作 Steam capsule A/B 的现代化分支。
- 可以把彩色路线卡系统合并进 N2-A。

## 图 3：R2-A 白金审计处刑局

![R2-A 白金审计处刑局](../originals/redline-r2-a-white-gold-bureau.png)

概念价值分：95.10

为什么是红线第一：

- 白金审计局比黑红哥特更高级，也更容易做 Steam 首屏差异。
- 成年女性执行官的角色吸引力比上一轮明显提升。
- 红 ink、纸质敌人、审计心脏、白色大厅形成统一世界观。
- 色情感被控制在服装和姿态上，没有破坏职业身份。

风险：

- 系统板可读性不如 R2-D。
- 红 ink 飞溅还需要更干净，避免被误读成血。

下一轮用途：

- 作为红线的市场首屏与角色基准。
- 与 R2-D 合并，而不是单独推进。

## 图 4：R2-D 医疗脉冲清算实验室

![R2-D 医疗脉冲清算实验室](../originals/redline-r2-d-medical-pulse-lab.png)

概念价值分：94.90

核心价值：

- 是红线里最能继续变成 UI/原型的方向。
- 心跳、扫描、判定、卡牌、清除都有明确系统含义。
- 明亮科幻医疗审计让红线摆脱暗黑哥特。
- 成人角色吸引力够，但审核风险低于赌场和霓虹街。

风险：

- 金融/清算局味道弱于 R2-A。
- 容易偏成科幻医疗游戏。

下一轮用途：

- 红线系统基准。
- 和 R2-A 做融合提示词：白金审计局空间 + 医疗脉冲 UI + 成年女性执行官。

## 图 5：N2-D 断钩清屏嘉年华

![N2-D 断钩清屏嘉年华](../originals/nightrail-n2-d-carnival-coupler-clear.png)

概念价值分：94.60

为什么大幅上升：

- 尺寸和文字先不扣后，它的“零号车票打孔 -> 断钩 -> 车厢飞离 -> 全屏清怪”非常强。
- 这是当前 8 张里最像 trailer 爆点的一张。
- 嘉年华灯光让清屏爆发不再只是黑烟和幽灵蓝，视觉更鲜艳。

风险：

- 不能单独作为主方向，容易变成宣传海报而不是系统。
- 需要 N2-A/N2-B 的卡牌系统支撑。

下一轮用途：

- 作为夜车的 signature payoff。
- 合并到 N2-A：日光装甲特急 + 嘉年华断钩清屏。

## 图 6：N2-C 红绒卧铺剧场

![N2-C 红绒卧铺剧场](../originals/nightrail-n2-c-velvet-sleeper-theatre.png)

概念价值分：94.30

核心价值：

- 成人擦边最明确，女列车长的市场点击力很强。
- 红绒、卧铺、剧场、车票扇让“夜车”更有情欲和危险感。
- 如果目标是测试轻度擦边是否提升点击，这张非常有参考意义。

主要问题：

- 又回到暗红哥特/剧场感，和本轮“跳出哥特”的目标冲突。
- 过强的红绒成人感可能弱化清怪爽感和系统感。

下一轮用途：

- 作为擦边强度上限，不做主方向。
- 可以把服装的肩颈、腿部、票扇武器设计合并到 N2-A。

## 图 7：R2-C 心跳赌桌判定室

![R2-C 心跳赌桌判定室](../originals/redline-r2-c-heartbeat-casino.png)

概念价值分：94.10

核心价值：

- 赌场、心跳、筹码、判定窗口的组合非常自然。
- 成人角色 market hook 强，且与“风险/债务/清算”题材相合。
- 适合做特殊章节、Boss 关或商人/高风险卡牌系统。

风险：

- 如果作为主方向，赌场 + 成人擦边 + 债务三者叠加会提高审核和地区风险。
- 容易从清算局动作卡牌变成赌场桌游幻想。

下一轮用途：

- 不做主线。
- 保留为红线的高风险房间、Boss 机制或卡牌事件。

## 图 8：R2-B 霓虹红账街

![R2-B 霓虹红账街](../originals/redline-r2-b-neon-debt-street.png)

概念价值分：93.90

核心价值：

- 战斗系统最街机，卡牌 combo、敌人标记、清屏节奏都清楚。
- 成人女性角色商业化明显。
- 能测试红线是否可以从办公室题材扩成都市夜街。

问题：

- 霓虹债务街不够独特，容易和泛赛博动作卡牌混在一起。
- 清算局、红笔、审计处刑的独有卖点被稀释。

下一轮用途：

- 只保留霓虹色彩和 combo 表达。
- 不作为红线主视觉路线。

## 当前决策

屏蔽“可直接交付执行”标准之后，推荐顺序变成：

1. **断轨夜车主线：N2-A 日光装甲特急。**
2. **夜车现代化备选：N2-B 霓虹末班地铁。**
3. **夜车爆点模块：N2-D 断钩清屏嘉年华。**
4. **红线主线融合：R2-A 白金审计处刑局 + R2-D 医疗脉冲清算实验室。**
5. **成人擦边上限参考：N2-C 红绒卧铺剧场。**
6. **红线特殊机制参考：R2-C 心跳赌桌判定室。**

## 下一轮建议

不要继续发散 8 个方向。现在应该收束成 3 张融合图：

1. `N2-A + N2-D`：日光/黄昏装甲特急 + 断钩清屏大招。
2. `N2-A + N2-B`：装甲特急 + 彩色路线票卡/现代 UI。
3. `R2-A + R2-D`：白金审计处刑局 + 医疗脉冲判定实验室。

如果只允许选一个继续，我仍建议先走 **断轨夜车：零号车票**，但具体美术路线从“哥特葬礼夜车”改成：

```text
黄昏装甲特急 + 成年女性列车长 + 物理车票卡牌 + 彩色路线系统 + 断钩清屏爆发
```
