问题：如果目标是轻度擦边、提升点击和留存，但不主动做成色情游戏，什么题材最适合？

## 结论

最适合的不是“纯美少女”，而是：

**成熟女性职业角色 + 危险关系 + 恐怖/异常/契约机制**

原因：

1. 轻度擦边需要女性角色，但女性角色不能只作为福利图存在。
2. 角色必须能自然进入卡牌、风险、局外成长和 run 内选择。
3. 商店首屏仍要 PG-13，不能依赖露骨素材吸引点击。
4. Steam tag 和推荐逻辑更需要准确定位；如果前排 tags 变成 `Sexual Content / Hentai / NSFW`，玩家期待会偏到成人内容供给，而不是 roguelite deckbuilder。

## 评分口径

| 维度 | 说明 |
|---|---|
| 擦边自然度 | 性感/暧昧是否来自题材，而不是硬贴 |
| 女性角色承载 | 是否能放入多个成年女性角色而不变成廉价后宫 |
| 机制贴合 | 是否自然解释卡牌、连击、牺牲、污染、局外成长 |
| 商店安全性 | capsule 能否 PG-13，同时仍有点击欲望 |
| 制作可控 | Web 原型阶段是否能少量资产先验证 |
| 差异化 | 是否不像普通黄油、普通地牢、普通卡牌 |

## 第一推荐：异常收容 / 夜班诊疗所

建议名方向：

- `夜班收容：体温档案`
- `异常诊疗所`
- `午夜解剖局`
- `Containment Nightshift`

一句话：

玩家是夜班收容员/异常医生，在一座会活化的地下诊疗所里，用规程卡、体温档案、器官样本和禁忌契约处理失控异常。

为什么最适合：

1. 女性角色可以是成年医生、调查员、黑市炼金师、魅惑异常、收容顾问，不必都做成偶像/学生。
2. 轻度擦边有合理语境：体温、诊疗、污染、检查、契约、夜班对话。
3. 卡牌机制可自然转译：规程卡、样本卡、诊断卡、禁忌治疗卡。
4. 擦边可以绑定风险：越危险/暧昧的契约，收益越高但污染越深。
5. 商店首屏可以 PG-13：成熟女医生 + 异常走廊 + 器官档案，不需要裸露。

适合的女性角色：

| 角色 | 擦边点 | 机制功能 |
|---|---|---|
| 夜班医生 | 白袍、手套、冷静检查、压迫感 | 治疗、抽牌、净化污染 |
| 黑市炼金师 | 危险交易、契约、身体代价 | 商店、诅咒卡、器官插槽 |
| 魅惑异常 | 声音、体温、视线污染 | 高收益卡、污染加速 |
| 收容调查员 | 战损制服、冷硬行动 | 初始角色、武器卡、移动能力 |
| 档案管理员 | 规则、记录、记忆交换 | 图鉴、永久成长、解锁 |

## 第二推荐：哥特血疗院 / 猎巫修道院

一句话：

玩家进入一座被血疗仪式污染的修道院，用圣印、血液、忏悔、猎巫契约组成卡牌链，镇压修女、吸血鬼、猎巫人和圣遗物异变。

优势：

- 哥特、血液、修女、女猎人天然有成熟性感张力。
- 血疗/忏悔/圣印非常适合卡牌和牺牲机制。
- 商店图可以很强：烛光、修道院、成年女猎人、血色卡牌。

风险：

- 容易滑向普通吸血鬼/修女擦边。
- 和 Vampire Survivors 的吸血鬼语义更近，原创性要靠“血疗/猎巫/卡牌仪式”拉开。

适合做法：

- 不叫吸血鬼，不做传统吸血鬼城堡。
- 重点放在“血疗院”和“仪式医疗”，保留身体/诊疗的机制解释。

## 第三推荐：魔女秘仪 / 女巫调查局

一句话：

玩家是调查局的契约执行人，进入魔女失控的仪式空间，用咒文卡、献祭材料、契约代价和女巫关系链完成封印。

优势：

- 女巫题材能自然承载成年女性角色和性感服装。
- 卡牌、仪式、颜色、费用、献祭都非常自然。
- 商店安全性最好：可以性感但不低俗。
- 比纯异常收容更容易做成风格化美术。

风险：

- 题材较常见，需要强机制名词和角色记忆点。
- 走廊压迫感不如收容设施直观，需要设计“仪式空间不断坍缩”。

## 第四推荐：赛博魅魔 / 地狱公司

一句话：

玩家是地狱企业审计员，在恶魔办公楼里用合同、债务、欲望、灵魂税组成卡牌 combo。

优势：

- 魅魔/恶魔契约天然适合擦边。
- 合同、债务、代价、税务非常适合卡牌文本。
- 容易做差异化和短视频传播。

风险：

- “魅魔”一出现，玩家很容易预期 adult-only 或黄油内容。
- 如果成人内容不够，可能被目标用户觉得“挂羊头”；如果太多，又偏离原型目标。

适合做法：

- 不把主角团队全做成魅魔。
- 把魅魔作为高风险 Boss/商人/契约来源，而不是题材标题的第一词。

## 不推荐作为第一主轴的方向

### 纯美少女异常小队

不是不能做，而是不适合作为第一阶段。

问题：

- 需要大量角色设定、立绘、皮肤、语音、剧情事件。
- 玩家会期待角色池和养成深度。
- Codex 原型验证会被美术生产问题吞掉。

可作为副结构：

- 只做 2-3 个成年女性核心角色。
- 不做抽卡式大角色池。
- 用“职业身份 + 危险关系”替代“美少女收集”。

### 妖怪旅馆 / 异界温泉

擦边自然，但战斗和走廊压迫不自然。更像经营/视觉小说/休闲 RPG，不适合 Vampire Crawlers-like 的压迫节奏。

### 仙侠狐妖 / 合欢禁术

中文市场可能很强，但全球 Steam 首屏风险更高：

- “合欢/双修”很容易把期待推向成人内容。
- 文化语汇需要本地化，否则海外玩家不一定理解。
- 如果不做足成人内容，反而会让期待错位。

可后续作为独立题材，不建议当前主线采用。

## 推荐最终融合方向

把当前 `夜班收容：器官档案` 调整为更适合轻度擦边的版本：

## `夜班收容：体温档案`

核心卖点：

- 不是后宫，不是黄油，是“危险诊疗 + 异常收容 + 卡牌构筑”。
- 成年女性角色承担职业和机制功能。
- 轻度擦边来自体温、污染、检查、契约、战损、夜班对话。
- 玩家追求的不是 CG，而是更强但更危险的卡牌路线。

机制映射：

| 系统 | 题材表达 |
|---|---|
| 卡牌费用 | 体温 / 心率 / 污染阈值 |
| 颜色 | 血液、神经、皮肤、骨骼、记忆 |
| gem slot | 器官插槽 / 样本皿 / 诊疗器械 |
| sacrifice | 献出样本 / 交换记忆 / 签署禁忌治疗同意书 |
| boss | 高危异常患者 / 魅惑实体 / 失控医生 |
| hub | 夜班诊疗所 / 档案室 / 黑市药柜 |
| meta progression | 执照等级、病例档案、角色信任、禁忌疗法 |

## 商店首屏构图建议

PG-13 capsule 可用：

- 成年女医生半身剪影，白袍/手套/红色病例夹。
- 背景是异常走廊和监控屏。
- 画面重点是眼神、手套、档案、红色警报，不是裸露。
- 副视觉可出现器官样本罐、污染血管、卡牌光效。

不要用：

- 露点、走光、泳装构图。
- 幼态脸、学生制服、未成年暗示。
- 以胸臀为中心的裁切。
- 直接把 `Sexual Content` 当第一屏卖点。

## 第一版原型角色配置

建议只做 3 个女性角色，不做大角色池：

1. **主角：夜班收容员**
   - 玩法：均衡卡组，适合教学。
   - 视觉：战损制服、污染痕迹。

2. **医生：白塔诊疗师**
   - 玩法：治疗、抽牌、净化，但会提高体温资源。
   - 视觉：白袍、手套、冷淡压迫感。

3. **异常：体温过高的第 7 号患者**
   - 玩法：高爆发、魅惑、污染加倍。
   - 视觉：成熟危险，不做露骨。

这样既有女性角色和擦边吸引，又不会变成美术产能黑洞。

## 当前推荐

选择：**异常收容 / 夜班诊疗所**。

具体名义：从 `夜班收容：器官档案` 进化为 `夜班收容：体温档案`。

这条路线最能同时满足：

- 有女性角色。
- 擦边自然。
- 不主动进入 adult-only。
- 机制和题材强绑定。
- Web 原型可控。
- 后续如果市场反馈需要，可以平滑升级到更成人向的 DLC/可选内容。
