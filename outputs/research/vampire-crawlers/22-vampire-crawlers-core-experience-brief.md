# Vampire Crawlers 主流核心体验 Brief

生成时间：2026-05-10 21:45 HKT  
用途：给后续主 agent / subagent / GDD 使用，避免题材策划偏离竞品核心体验。  
输入：Steam 评测、媒体评测、YouTube 短片段、三条长 VOD 前三小时材料、`14-first-3-hours-experience-analysis.md`。

## 一句话核心

`Vampire Crawlers` 的核心不是“卡牌题材”或“地牢题材”，而是：

**用 roguelike deckbuilder 的构筑选择，驱动 Vampire Survivors 式的快速清怪、连锁爆发和一局接一局的成长爽感。**

任何原创题材都必须先服务这个体验，轻度擦边和女性角色只能增强它，不能替代它。

## 核心体验分层

| 层级 | 玩家感受到什么 | 设计意义 |
|---|---|---|
| 即时战斗层 | 怪物快速压近，玩家连续清怪、回血、升级、抽牌 | 必须有高频击杀和清场反馈 |
| 卡牌构筑层 | 抽牌、费用、颜色、combo、升级、socket/gem 影响战斗节奏 | 卡牌不是奖励菜单，而是战斗引擎 |
| 空间压迫层 | 伪 3D / 第一人称走廊让怪潮有距离和逼近感 | 不能做成纯平面菜单或静态战斗 |
| 爆发爽感层 | build 从弱到强，后期出现一串斩杀、清场、翻盘 | 10 分钟原型必须有一次大爆发 |
| 局外成长层 | 商店、hub、地图推进、永久升级、解锁让玩家继续下一局 | 失败也要留下明确收益 |

## 前三小时体验曲线

| 阶段 | 核心体验 | 对原创项目的要求 |
|---|---|---|
| 0-15 分钟 | 快速理解“幸存者 + 卡牌 + 地牢”的差异化 | 开场 2 分钟内必须杀怪，不要先讲设定 |
| 15-45 分钟 | 第一套 build 出现，玩家感到卡牌能救命 | 第 5-8 分钟要出现一次构筑翻盘 |
| 45-90 分钟 | boss、奖励、卡牌升级和地图推进形成中段压力 | 中段要有明确的精英/首领节点 |
| 90-180 分钟 | gem slots、牺牲、reroll、永久成长等 meta 系统显形 | 第三小时要看到“下一局更强”的目标 |

## 主流核心爽点

### 1. 快速斩杀怪物

这是一号约束。

玩家需要看到：

- 怪物不断涌来。
- 攻击不断触发。
- 敌人快速死亡。
- 奖励频繁掉落。
- 数字、音效、光效和 UI 持续反馈。

如果题材让玩家主要在“检查、谈话、经营、记录”，就已经偏离。

### 2. 走廊压迫

Vampire Crawlers 的伪 3D / Unity 表现不是为了炫技术，而是为了让怪物逼近更有压迫感。原创原型需要保留：

- 狭窄或半狭窄空间。
- 怪物从视野前方/两侧逼近。
- 玩家需要边走边处理卡牌/攻击。
- 精英和 boss 形成时间边界。

### 3. 卡牌是战斗引擎

VOD 证据显示卡牌系统包含：

- draw card
- mana / cost
- color
- combo
- upgrade
- socket / gem slot
- sacrifice correct color
- reroll

这说明卡牌不是“升级三选一”，而是局内战斗节奏的一部分。原创项目必须至少验证：

- 抽牌。
- 费用。
- 连锁。
- 卡牌升级或插槽。
- 高风险高收益牌。

### 4. 构筑翻盘

最重要的中期爽感是：

1. 局势快崩。
2. 玩家抽到关键牌或拿到关键升级。
3. 组合突然成立。
4. 屏幕清场，玩家获得“我 build 对了”的反馈。

原创题材必须支持“将要崩盘 -> 构筑救场”的叙事和反馈。

### 5. 高频微奖励

玩家被持续喂给：

- XP。
- 新卡。
- 升级。
- 金币/资源。
- 商店机会。
- 解锁目标。
- boss 奖励。

轻度擦边内容不能打断这个节奏。它最好作为奖励包装和风险选择出现，而不是长文本事件。

### 6. 局外下一局动机

Vampire Crawlers 的 IP 光环可以让玩家忍受慢热。原创项目没有这个红利，所以必须更早给出：

- 下一局起始牌。
- 角色/武器/卡牌解锁。
- build 路线预告。
- 视觉/语音/档案奖励。
- 明确的“再开一把会变强”。

## 已知痛点

媒体和用户反馈中反复出现的风险：

| 痛点 | 对原创项目的启发 |
|---|---|
| 前期可能重复 | 第一局就要给 2-3 条有效 build 路线 |
| 永久成长可能显慢 | 原型阶段要更早展示解锁收益 |
| UI 可读性/掌机可读性 | 卡牌、关键词、费用、伤害必须清楚 |
| 移动或手柄菜单可能繁琐 | 卡牌选择不能让动作节奏断太久 |
| 品牌红利太强 | 原创项目必须更早展示差异化和爆发 |

## 对轻度擦边题材的硬约束

女性角色和擦边元素必须满足。注意：这里的“女性角色”不等于“女性主角”。`Vampire Crawlers` 是多英雄、多构筑入口的游戏，原创项目也不应被锁死为单女性主角。女性吸引力可以来自英雄、怪物、Boss、商人、契约对象、异常同伴或局外 NPC。

1. **战斗优先**：女性角色最好能参与战斗系统，可以是猎人、魔女、处刑官、契约执行人、异常敌人、Boss、商人或高风险卡牌来源，而不是只在 hub 里说话。
2. **动词优先**：斩、爆、焚、封、处决、清场，优先于诊疗、记录、复查、经营。
3. **擦边服务 build**：性感/暧昧选择必须对应卡牌收益、污染风险、爆发路线或局外解锁。
4. **事件短促**：事件只能作为 run 后或节点奖励，不能把游戏变成视觉小说。
5. **首屏必须有怪潮**：商店页和原型第一屏要让玩家知道这是动作清怪游戏。

## 女性吸引力的可用位置

| 位置 | 用法 | 优点 | 风险 |
|---|---|---|---|
| 可选英雄 | 成年女猎人/魔女/处刑官作为可选构筑入口 | 角色直接绑定卡组和战斗风格 | 资产成本提高，容易变成角色池压力 |
| 女性 Boss | 魅惑异常、血月圣女、契约魔女作为阶段 Boss | 强记忆点，适合短视频和关卡目标 | 不能做成受害/俘虏成人化 |
| 女性怪物潮 | 低阶怪物包含女妖、血仆、镜中人等 | 擦边进入战斗，不依赖主角性别 | 需要避免全怪物性化导致低俗 |
| 商人/契约对象 | 黑市契约师、血术商人提供高风险卡牌 | 很适合 risk/reward 和局外留存 | 不能把 hub 对话拉太长 |
| 异常同伴 | 击败后成为卡牌/召唤物/诅咒源 | 支持多英雄和多 build | 需要明确机制，不做纯收藏图 |

推荐结构：**男女/中性英雄混合 + 女性高记忆 Boss/商人/异常同伴**。这样不会把项目锁成“女性主角游戏”，但仍然保留擦边市场吸引力。

## 给 subagent 的方向约束

评估任何题材时，请先回答：

1. 这个题材能不能在 10 秒内让玩家看到“我正在快速杀怪”？
2. 女性吸引力是否能直接参与战斗、发卡、触发爆发、成为高危 Boss 或局外高风险契约来源？
3. 擦边元素是否能转化成 risk/reward，而不是纯文本福利？
4. 是否有清晰的敌人死亡反馈和连锁清场画面？
5. 是否会误导成模拟经营、视觉小说或纯黄油？

如果答案不满足，题材再性感也不应作为主方向。

## 当前修正后的推荐方向

基于这个 brief，当前不再推荐 `夜班诊疗所/体温档案` 作为主方向。

更合理的候选是：

1. **血月猎巫：修道院清算**
   - 强视觉、强斩杀、女性猎人/修女/血月 Boss。
2. **夜班猎杀：异常清除令**
   - 保留异常收容，但主角是清除员，核心是清怪和处决。
3. **魔女处刑局：禁忌合同**
   - 魔女/契约/处刑，轻度擦边更安全，系统感强。

它们都必须以“快速清场 + 卡牌 build 爆发”为第一目标。
