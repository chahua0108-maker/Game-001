## 核心修正

此前 `夜班诊疗所/体温档案` 的问题已经确认：

**它的擦边自然，但战斗动词太弱。**

本项目不能先从诊疗、体温、病历、关系事件出发。它必须先从 Vampire Crawlers 的核心体验出发：

1. 快速清怪。
2. 走廊压迫。
3. 卡牌构筑。
4. build 爆发。
5. 局外解锁。

轻度擦边只作为角色吸引、感官张力和 risk/reward 放大器。

## 两个子 agent 的关键结论

### 风险审核子 agent

最重要结论：

1. “战斗优先、擦边调味”比 adult 主轴、诊疗主轴、关系事件主轴都更稳。
2. 成年女性猎人/魔女/修女快速清怪，成人误读风险低于“夜班诊疗所/体温档案”。
3. 风险最稳排序：`夜班猎杀：异常清除令` > `魔女处刑局：禁忌合同` > `血月猎巫：修道院清算`。
4. 夜班诊疗所只适合做 hub、档案、升级、短事件层，不适合做主轴。
5. 风险会从成人误读转移到暴力表达：处决、血液、战损、宗教服饰必须风格化。

### 竞品子 agent

最重要结论：

1. `Hell Maiden` 最贴“快速清怪 + 卡牌 + 女性角色 + 非 adult-only 倾向”。
2. `Nunholy` 最贴“哥特/吸血鬼/女性猎人/轻擦边”的题材壳。
3. `HoloCure` 是非 adult-only 的清怪反馈密度标杆。
4. `Hades II` 适合学习性感角色魅力、短促事件和局外关系留存。
5. `Bayonetta` 适合学习“性感来自战斗姿态和处决反馈”，不是静态暴露。

## 推荐主方向：夜班猎杀：异常清除令

### 一句话

多名清除员在异常设施夜班中执行清除令，用规程卡、处决卡、污染契约和武器卡快速斩杀怪潮；女性吸引力可以来自可选英雄、高危 Boss、怪物、商人和异常同伴，而不必强行锁成单女性主角。

### 为什么它现在排第一

1. 保留异常收容的差异化，但不再做诊疗模拟。
2. “清除令/猎杀/处决”直接承诺战斗爽感。
3. 女性吸引力可以进入多位置：可选英雄、Boss、怪物潮、商人、异常同伴，不只是 hub NPC。
4. 擦边来自战损、污染、契约、低语、危险 Boss，而不是成人事件。
5. 审核风险低于修女/血月，也低于诊疗/身体检查误读。

### 核心循环

`开门 -> 怪潮涌入 -> 出牌斩杀 -> 连锁清场 -> 污染/契约变化 -> 奖励/升级 -> 下一扇门`

### 角色结构

| 角色 | 作用 | 擦边表达 | 机制 |
|---|---|---|---|
| 夜班清除员 | 可男可女的基础英雄 | 战损制服、污染纹路、冷硬台词 | 均衡战斗卡 |
| 异常处刑官 | 女性高爆发可选英雄 | 黑色手套、处决仪式 | 标记、斩杀、连锁处决 |
| 叛逃异常体 | 女性 Boss / 可解锁异常同伴 | 魅惑、低语、危险靠近 | 高收益污染卡 |
| 黑市契约师 | 女性商人 / 契约来源 | 危险交易、短句压迫 | 诅咒卡、契约升级 |
| 血仆群 / 镜中人 | 女性怪物潮的一部分 | 剪影、红线、凝视 | 标记、聚怪、爆裂连锁 |

### 首版原型必须有

- 每 3-5 秒一次小清场。
- 第 5-8 分钟一次屏幕级连锁处决。
- 至少 30 张卡，其中 10 张直接杀怪或清场。
- 至少 3 种敌人死亡反馈：封印爆裂、红线切断、污染燃烧。
- 擦边反馈只出现在战斗状态、角色语音、卡牌文案和局外短事件中。

## 备选强方向：血月猎巫：修道院清算

### 一句话

猎巫队伍在血月修道院中用圣印、血刃、忏悔链和禁忌誓约卡快速处决怪潮；女性角色可以是女猎人、血月 Boss、血仆怪潮、忏悔商人或被解锁的异常同伴。

### 为什么有吸引力

- 商店首屏最强：血月、修道院、女猎人、怪潮、卡牌。
- `Nunholy` 等案例证明哥特女性猎杀吸血鬼的壳有市场识别。
- 擦边自然来自血契、禁忌誓约、战损、忏悔、危险亲密。

### 风险

- 宗教服饰、修女、血液、处决容易被 fetish 化或 gore 化。
- 必须架空宗教，减少现实符号。
- 适合做强视觉方向，但第一版原型需要更高美术要求。

## 稳健方向：魔女处刑局：禁忌合同

### 一句话

多名契约执行人进入合同走廊，用咒印、违约条款、处刑卡和禁忌合同清算失控恶魔；女性吸引力主要来自魔女 Boss、契约商人、可选魔女英雄和异常委托人。

### 为什么稳

- 魔女题材对轻度擦边最安全，不天然承诺 adult-only。
- 契约、条款、印章、代价与卡牌机制天然同构。
- 文案、UI 和卡牌效果可以先撑起原型。

### 风险

- 清怪画面不如猎巫/异常清除直观。
- 如果太偏合同文案，会再次滑向“系统/文本游戏”。
- 必须加入强战斗视觉：印章爆炸、符文切割、违约处刑、合同燃烧。

## 竞品体验约束

后续任何 GDD 必须先过这张表：

| 约束 | 最低标准 |
|---|---|
| 快速清怪 | 原型开局 30 秒内开始杀怪 |
| 卡牌引擎 | 抽牌、费用、combo 至少一条跑通 |
| 走廊压迫 | 怪物必须从空间中逼近，不只是菜单结算 |
| build 爆发 | 第 5-8 分钟出现一次连锁清场 |
| 擦边绑定 | 女性英雄/Boss/怪物/商人/同伴中的擦边元素必须改变卡牌、污染、状态、角色能力或局外解锁 |
| 平台边界 | 首屏 PG-13，游戏内成熟暗示，非 adult-only 主轴 |
