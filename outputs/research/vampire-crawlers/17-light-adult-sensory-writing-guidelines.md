# 轻度擦边的视觉、声音与文案准则

生成时间：2026-05-10 20:55 HKT  
适用方向：`夜班收容：体温档案` / 异常收容 + 夜班诊疗所 + 卡牌构筑  
尺度：S0 成熟但安全 + 少量 S1 轻度擦边，不主动进入 adult-only。

## 核心判断

擦边不是图片问题，而是**感官系统问题**。

如果只靠立绘暴露，会廉价，也容易把玩家期待带偏。更稳的方式是让视觉、声音、文案和机制共同制造“危险亲密感”：

- 视觉负责第一眼吸引。
- 声音负责贴近感和紧张感。
- 文案负责暧昧、留白和角色关系。
- 机制负责把诱惑变成 risk/reward。

也就是说，玩家感受到的不是“这里有福利图”，而是“这个角色/异常很危险，但我想靠近，因为靠近能变强”。

## 总体表达边界

| 可用 | 不用 |
|---|---|
| 成年角色、职业服装、战损、污染痕迹、手套、白袍、体温、脉搏、契约、夜班、禁忌治疗 | 未成年/幼态角色、校园未成年暗示、露点、走光、明确性场景、真实人物露骨图像 |
| 暧昧但克制的台词、危险邀约、代价选择、可跳过事件 | 直接粗俗性描写、强制性/非自愿语境、把玩家羞辱当主要卖点 |
| 低频贴近声、心跳、呼吸、衣料、手套、监护仪、门锁、纸张 | 成人呻吟、露骨音效、长时间 ASMR 黄油化 |
| 机制上的诱惑：高收益、高污染、高失控 | 纯 CG 解锁、纯皮肤收集、成人内容替代玩法 |

## 视觉层

视觉目标：**成熟、危险、可记忆，但商店首屏仍可 PG-13。**

### 可用元素

| 元素 | 用法 |
|---|---|
| 白袍 / 手套 | 强化诊疗、控制、距离感；手套可作为 UI 指针和卡牌图标 |
| 战损制服 | 表达 run 内压力和污染变化，不做裸露中心构图 |
| 红色病例夹 / 体温计 / 监护仪 | 让“体温档案”成为识别符号 |
| 皮肤污染纹路 | 替代直接裸露，表现危险与代价 |
| 眼神与半身剪影 | 用表情和构图制造吸引，而不是靠露出 |
| 样本罐 / 器官插槽 | 连接卡牌机制和身体主题 |

### 构图规则

1. 首屏图不以胸臀裁切为中心。
2. 角色必须有明确职业身份或异常身份。
3. 性感来自姿态、眼神、材质和风险，而不是暴露面积。
4. 角色年龄感必须明确成年。
5. 战斗截图优先展示卡牌、走廊、敌潮和 UI，不让擦边遮蔽玩法。

## 声音层

声音目标：**近、冷、危险，而不是露骨。**

### 可用声音语言

| 场景 | 声音设计 |
|---|---|
| 抽牌 | 纸张摩擦 + 轻微心跳 |
| 打出禁忌卡 | 手套拉紧声 + 监护仪短促报警 |
| 污染上升 | 低频脉冲 + 近距离呼吸感，但不做呻吟 |
| 魅惑异常出现 | 近距离耳语质感 + 房间混响 + 玻璃震动 |
| 夜班诊疗所 hub | 荧光灯、电流、远处高跟鞋/脚步、病例纸翻动 |
| 角色关系事件 | 低声、短句、停顿，不做长段成人语音 |

### 声音边界

- 不做明显性行为音效。
- 不做“呻吟即奖励”的反馈。
- 不让声音比战斗反馈更抢戏。
- 心跳/呼吸必须服务紧张和污染，不服务露骨联想。

## 文案层

文案目标：**暗示多于直说，代价多于挑逗。**

擦边文案的核心不是“说色情词”，而是把玩家带到一个危险选择前：

- 你可以靠近。
- 你知道靠近不安全。
- 你仍然想靠近，因为它能改变 build。

### 句式原则

| 原则 | 示例方向 |
|---|---|
| 用身体信号替代露骨词 | 体温、脉搏、呼吸、皮肤、指尖、血样、记忆 |
| 用契约替代直白关系 | 签署、同意、代价、交换、禁忌条款 |
| 用职业语汇包住暧昧 | 诊疗、复查、隔离、记录、样本、异常反应 |
| 用留白制造张力 | “她没有继续解释，只把手套放在你的病例上。” |
| 用机制收束文案 | 每段暧昧都要落到卡牌、污染、解锁或代价 |

### 卡牌命名方向

| 类别 | 命名方向 |
|---|---|
| 诊疗卡 | `复温注射`、`脉搏校准`、`白塔手套`、`静默复查` |
| 污染卡 | `体温越界`、`皮肤记忆`、`红线反应`、`异常潮红` |
| 契约卡 | `禁忌同意书`、`第七条代价`、`夜班签署`、`黑市复诊` |
| 魅惑卡 | `低声召回`、`视线污染`、`靠近指令`、`危险安抚` |
| 防御卡 | `隔离帘`、`冷光消毒`、`镇定剂量`、`病例封存` |

### 不建议的命名

- 直接粗俗性词。
- 明确性行为描述。
- 只靠身体部位做卖点的卡名。
- 含未成年、校园、教师学生等容易误读的语汇。

## UI 与系统文案

擦边不仅在剧情，也可以在系统里。

| 系统 | 普通叫法 | 擦边但安全的叫法 |
|---|---|---|
| HP | 生命值 | 体征稳定度 |
| Mana | 法力 | 体温阈值 |
| Curse | 诅咒 | 污染反应 |
| Upgrade | 升级 | 复诊 / 校准 / 追加疗程 |
| Sacrifice | 献祭 | 样本提交 / 代价签署 |
| Shop | 商店 | 黑市药柜 |
| Codex | 图鉴 | 病例档案 |
| Reroll | 重掷 | 重新诊断 |

这种改名能让系统一直在题材里，不需要额外塞成人剧情。

## 事件文案结构

每个轻度擦边事件建议控制在 4 步：

1. **危险接触**：角色提出一个暧昧但合理的职业/异常行为。
2. **明确选择**：玩家选择靠近、拒绝或转移代价。
3. **机制结果**：获得强卡、污染、体温变化、关系推进或解锁。
4. **留白收尾**：一句短文案，不解释过多。

示例结构：

| 步骤 | 内容目标 |
|---|---|
| 触发 | 夜班医生要求你留下新的血样，因为上一次样本“太热” |
| 选择 A | 接受复查：获得抽牌卡，污染 +1 |
| 选择 B | 拒绝复查：获得护盾，关系不变 |
| 选择 C | 签署禁忌同意书：获得稀有卡，污染 +3 |
| 收尾 | 她把病例合上，没有看你，只说“下次别来得这么晚。” |

## 角色台词方向

台词要短，像战斗 UI 的调味，不要像黄油长对话。

### 夜班医生

- “体温又越线了。坐下。”
- “别动，污染会顺着脉搏往上爬。”
- “这不是治疗，是校准。”
- “你可以拒绝，但病例会记住。”

### 黑市炼金师

- “便宜的药没有副作用，贵的才有。”
- “签这里。别问第七条。”
- “你想要更强，还是更干净？”

### 魅惑异常

- “靠近一点，我会把路让出来。”
- “你听见的是心跳，不一定是你的。”
- “抽下一张。看看你还剩多少理智。”

这些台词有擦边张力，但都能落回机制。

## 留存设计

轻度擦边对留存的价值主要在局外，不在局内。

| 留存点 | 设计 |
|---|---|
| 每局后关系推进 | 根据本局污染/卡组选择触发不同短事件 |
| 档案解锁 | 解锁角色病例、异常记录、卡牌背景 |
| 视觉变化 | 角色战损/污染阶段随 build 路线变化 |
| 语音变化 | 高污染状态下，角色台词更近、更短、更危险 |
| 下一局诱因 | 角色事件给下一局专属起始牌或禁忌选择 |

注意：不要让玩家“刷成人内容”，而是让玩家“为了 build 和角色关系再开一局”。

## 原型阶段落地范围

第一版只做这些：

1. 3 个成年女性角色的文字设定和 1 张占位立绘方向。
2. 30 张卡牌名与效果，其中 8-10 张带轻度擦边语汇。
3. 6 个夜班事件，每个 4 步结构。
4. 5 种声音占位：抽牌、手套、心跳、监护仪、低语。
5. 1 条污染/体温资源，直接影响卡牌收益和角色事件。

暂时不做：

- 成人 CG。
- 角色池。
- 露骨语音。
- 大量皮肤。
- 明确 adult-only 内容。

## 质量检查清单

每个擦边元素上线前检查：

1. 角色是否明确成年？
2. 是否有职业/异常身份，而不只是身体展示？
3. 是否能绑定一个机制结果？
4. 去掉这段擦边后，玩法是否仍成立？
5. 它是否会让商店首屏或 community item 违反 PG-13/全年龄边界？
6. 它是否会让玩家误以为这是成人游戏？
7. 它是否能提高下一局动机，而不仅是一次性点击？

只有全部通过，才进入正式 GDD。
