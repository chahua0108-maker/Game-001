## 方向一：深井电梯：第九十九层 / Shaftfall: Floor 99

### 1. 中英暂名

**深井电梯：第九十九层 / Shaftfall: Floor 99**

### 2. 一句话 hook

在一台不断坠落的老式铁栅电梯里，把手牌变成楼层按钮、门闸和配重机关，在每一次“叮”的开门声中清掉从四面楼层涌入的怪潮。

### 3. 首屏 capsule / GIF 画面

**Capsule：** 一台黄铜铁栅电梯悬在无底竖井中央，门外每一层平台都挤满白眼怪物；成年女电梯员一手拉闸、一手按亮 13、27、66 三枚发光楼层按钮，电梯门像铡刀一样切开冲进来的怪潮，金币、火花和 XP 碎片从竖井里向上喷。

**10 秒 GIF：**

1. `0-2s`：电梯灯闪烁，楼层指针从 `42` 掉到 `41`，四扇铁栅门同时响起“叮”。
2. `2-4s`：怪潮从正前方走廊、左右维修栈桥和头顶电缆爬入，玩家抽到三张“楼层按钮牌”。
3. `4-7s`：玩家支付电力，打出 `关门剪切 -> 超载下坠 -> 配重反冲`，门闸切怪、电梯急坠、怪物被压成亮色碎片。
4. `7-10s`：新卡插入按钮面板，`99F` 亮起，整台电梯反向冲顶，屏幕上下两端怪潮同时被卷进电缆风暴。

这张图和现有候选拉开的点：不是线、票、光束或设施，而是**一个人人秒懂的封闭电梯盒子 + 垂直竖井怪潮 + 楼层按钮卡牌引擎**。

### 5. 10 秒杀怪承诺

开局 10 秒内玩家必须完成一次：

`抽牌 -> 按亮楼层按钮 -> 电梯门切开第一排怪 -> XP 火花吸回 HUD -> 新楼层警报亮起`

即使没有 build，基础牌也能让玩家看到“怪潮逼近、出牌、清怪、奖励”。

### 6. 第 5-8 分钟 build 大爆发

**爆发名：99F 反坠冲顶。**

第 5-8 分钟，玩家第一次凑齐：

`同色楼层三连 + 超载层数 6+ + 配重反冲 socket`

效果：

1. 电梯先向下急坠 13 层，碾碎下方竖井怪潮。
2. 配重链被 Boss 拉断，触发反向冲顶。
3. 已插 socket 的按钮依次重打，形成二次连锁。
4. 屏幕顶部、底部、左右门同时清场，楼层指针短暂显示 `99F`，掉落一张金色按钮升级。

玩家感受是：刚才快被四门怪潮挤死，但我把按钮顺序和超载倍率做对了，整台电梯变成杀怪炮膛。

### 7. 核心系统映射

| 系统 | 题材映射 | 战斗含义 |
|---|---|---|
| 抽牌 | 电梯控制面板弹出可用按钮、拉闸、维修指令 | 每回合/每几秒给玩家新的机械动作 |
| 费用 | 电力 / 载重压力 | 高费牌会加速超载，也提高爆发倍率 |
| 颜色/类型 | 楼层色：住宅黄、机房蓝、宴会红、维修绿、禁层紫 | 同色楼层连按触发连锁；不同类型对应门、坠落、配重、救援 |
| combo | 连续按相邻楼层、同尾数楼层、上下反向楼层 | 例如 `13 -> 23 -> 33` 触发竖井横扫 |
| upgrade | 按钮镀层、楼层铭牌升级、门闸齿升级 | 提高伤害、减费、改变开门方向 |
| socket/gem | 把保险丝、齿轮、铃芯插进按钮孔位 | 让某张牌附带二次效果，如吸血、眩晕、复制 |
| 牺牲 | 拆掉一个已亮按钮，释放备用电力 | 牺牲正确颜色可触发紧急清场；牺牲错色会断电 |
| reroll | 手动重排楼层面板 / 召唤下一班电梯 | 改变当前手牌，但增加下一波怪潮速度 |

### 8. 伪 3D / 走廊表现方案

主视角固定在电梯内部，玩家面向铁栅门和楼层面板。空间不是横向长走廊，而是**竖井走廊**：

- 正前方：每次开门出现短走廊，怪物从楼层深处冲向电梯。
- 左右侧：维修栈桥和半开门缝涌入侧向怪潮。
- 上下方：电缆、配重、天花板井道形成垂直威胁，飞行怪和爬行怪从视野边缘进入。
- 电梯移动用背景楼层快速滑动、载重针抖动、门框透视变化表现，不需要真 3D 大地图。

### 9. 成年女性英雄 / Boss / 商人 / 同伴 / 怪潮位置

1. **玛格达，午夜电梯员 / Magda, Midnight Lift Operator**：可选英雄。起始牌偏门闸和紧急制动；她的被动是每次救回濒死局面后生成一张低费按钮牌。
2. **赫萝丝，顶层住户 / Heloise of the Penthouse**：第 8-10 分钟 Boss。她不站桩，而是在不同楼层开门发号施令；击败后可把一张 Boss 楼层牌加入牌池。
3. **维拉，检修夹层商人 / Vera in the Service Gap**：商人。出售高风险保险丝 socket；代价是把一张“断电楼层”诅咒洗入牌组。

她们都进入战斗或发卡机制，不作为静态 hub 福利。

### 10. 10 张核心卡牌或关键词：按 combo 链排列

1. **开门诱饵 / Open-Door Bait**：指定一侧开门，吸引怪潮集中。
2. **铁栅剪切 / Grille Shear**：关门造成横向斩击。
3. **楼层连按 / Floor Chain**：打出同色按钮时抽 1。
4. **载重红针 / Overload Needle**：本轮每杀 10 个怪增加 1 层超载。
5. **急坠十三层 / Thirteen-Floor Drop**：按超载层数造成竖向碾压。
6. **配重反冲 / Counterweight Recoil**：下坠后复制上一张门牌。
7. **保险丝 socket / Fuse Socket**：给按钮牌插入元素效果。
8. **紧急制动 / Emergency Brake**：取消一次致命开门，同时对贴脸怪造成眩晕。
9. **拆钮供电 / Button Sacrifice**：牺牲一张已升级按钮，恢复电力并触发同色爆炸。
10. **99F 反坠冲顶 / Floor 99 Reversal**：终端 combo，按本局已触发楼层数连锁清场。

### 11. 轻度擦边边界

- **商店首屏 PG-13**：成年电梯员、顶层住户、检修商人的服装可以是晚礼服、制服、维修束带、金属饰品，但不出现裸露重点、性行为暗示、受害化姿态。
- **游戏内成熟危险感**：靠窄空间、贴脸怪潮、低语楼层广播、危险角色交易制造张力。
- **擦边服务系统**：角色诱惑不是成人内容，而是高风险卡牌来源，例如“顶层邀约”给强力 Boss 牌，同时增加超载和禁层怪潮。

### 12. 为什么能超过红线清算局或断轨夜车

**超过红线清算局的市场稳健性：**

- 电梯是全球玩家共同经验，1 秒可读，不依赖合同、清算、条款等语言理解。
- “门开怪来、按按钮、门切怪”比红线/签章更直接动作化，不容易滑成文本系统。
- 封闭电梯天然适配 Steam capsule：画面中心只有一个盒子、一面按钮板、一群怪，缩略图损失小。

**接近并局部超过断轨夜车的设计记忆：**

- 夜车强在车厢推进，电梯强在垂直楼层和按钮面板；后者更容易做成一个 10 分钟原型。
- 每次“叮”都是短视频钩子，比列车长段移动更适合 3-5 秒循环。
- 楼层数字、按钮 socket、载重针和门闸都是可触摸工具，不弱于车票打孔的机械感。

### 13. 最大失败风险和一句补救

**风险：** 电梯空间太小，可能被误解为固定炮塔小游戏，缺少 Vampire Crawlers 的推进感。

**补救：** 把“楼层下降/冲顶”做成持续推进的 run 进度，并让每 30-45 秒出现新楼层走廊、侧门和竖井事件，确保玩家一直在移动、清怪、升级。

## 方向二：活字屠版厂：墨潮印杀 / Inkpress Cathedral: Killprint Run

### 1. 中英暂名

**活字屠版厂：墨潮印杀 / Inkpress Cathedral: Killprint Run**

### 2. 一句话 hook

在一座像大教堂一样轰鸣的活字印刷厂里，把手牌排成金属字模，让印刷机一下一下把墨潮怪物压成咒文、弹壳和升级材料。

### 3. 首屏 capsule / GIF 画面

**Capsule：** 巨型黑色印刷机占据画面中央，前方传送带上挤满由墨水、纸皮和排版碎骨组成的怪潮；成年女排字师把三张发光字模牌塞进金属排版框，压印滚筒砸下，整条走廊被一行燃烧大字扫空：`CUT / BURN / REPEAT`。

**10 秒 GIF：**

1. `0-2s`：印刷厂走廊深处涌来墨潮怪，地面纸带向玩家脚下卷。
2. `2-4s`：手牌变成金属字模，玩家把 `CUT`、`INK`、`PRESS` 三个字模拖进排版框。
3. `4-7s`：印刷机轰然下压，字模变成三段攻击：纸刃横切、黑墨爆裂、滚筒碾压。
4. `7-10s`：被杀怪物在纸带上留下新字母，排版框解锁一个 socket，下一轮能把整句咒文复制两遍。

这张图和现有候选拉开的点：不是交通、债务、光学或收容，而是**工业印刷机 + 活字排版 + 墨潮怪物 + 字句连锁清场**。玩家 1 秒能复述“印刷机压怪”，10 秒能看懂“出牌排字，机器清场”。

### 5. 10 秒杀怪承诺

开局 10 秒内玩家必须完成一次：

`抽到三枚字模 -> 排入三格版框 -> 踩下压印踏板 -> 前排墨怪被压成字母碎片 -> 掉落新字模/XP`

即使玩家不懂任何系统，也能从画面看懂“我正在用印刷机杀怪”。

### 6. 第 5-8 分钟 build 大爆发

**爆发名：整版连印 / Full-Page Overprint。**

第 5-8 分钟，玩家第一次凑齐：

`同墨色三词 + 版框 socket + 连印滚筒`

效果：

1. 当前排版句子先正常压印一次。
2. socket 中的金属铅芯把第一个动词复制到全句每个词前。
3. 连印滚筒把整句横向铺满走廊，形成 3-5 道纸刃/墨爆/钉印连锁。
4. 屏幕上所有怪物死亡后被印成“校样页”，玩家从校样页里三选一升级。

玩家感受是：原本只是 `CUT INK PRESS` 三个小动作，build 成形后变成满屏命令句，印刷机像机关枪一样连续压怪。

### 7. 核心系统映射

| 系统 | 题材映射 | 战斗含义 |

| reroll | 重排字盘 / 洗掉校样纸 | 换手牌，但可能印出一个“错字怪”加入下一波 |

### 8. 伪 3D / 走廊表现方案

主视角在印刷机末端，玩家看向一条由纸带、传送带和铅字轨道组成的长走廊：

- 前方：墨潮怪沿纸带冲来，距离感由纸带刻度和滚筒阴影表现。
- 左侧：字盘和排版框占 UI 区，手牌像实体字模被推入框内。
- 右侧：墨槽、切纸刀、校样纸显示资源、冷却、下一波提示。
- 战斗攻击沿走廊发射：纸刃横扫、墨浪溅射、滚筒虚影向前碾压、钉印从地面弹起。
- Boss 可从走廊深处撕开巨幅海报入场，或者把玩家排好的句子污染成反义词。

### 9. 成年女性英雄 / Boss / 商人 / 同伴 / 怪潮位置

1. **伊芙琳，夜班排字师 / Evelyn, Night Typesetter**：可选英雄。起始牌偏低费动词和重排；被动是每印出完整三词句就保留一个字母到下轮。
2. **玛拉，墨后校对员 / Mara, Queen Proofreader**：Boss。她会把玩家句子中的一个词改成危险反义词，例如 `CUT` 变 `FEED`；击败后解锁“校对”类防御/复制牌。
3. **赛琳，铅芯走私商 / Celine, Lead-Core Broker**：战斗内商人。她在短暂停机窗口出现，出售 socket 铅芯；代价是往牌组洗入一张禁字牌。

她们都通过起始牌、Boss 改词、socket 交易进入构筑链，不依赖长对话或图库奖励。

### 10. 10 张核心卡牌或关键词：按 combo 链排列

1. **CUT 字模 / CUT Type**：基础横切，动词位。
2. **SWARM 字模 / SWARM Type**：指定怪潮目标，名词位。
3. **PRESS 踏板 / PRESS Pedal**：机器位，压印当前句子。
4. **BLACK INK / 黑墨**：低费墨色，杀怪回墨。
5. **RED INK / 朱墨**：高伤墨色，增加过热。
6. **版框 socket / Chase Socket**：给一个词位嵌入铅芯。
7. **双印滚筒 / Double Roller**：复制当前句最后一个词。
8. **熔字回铅 / Melt Type**：牺牲字模，恢复费用并提高下一次压印倍率。
9. **校样重排 / Proof Reroll**：重抽手牌；若抽到禁字，生成错字怪。
10. **整版连印 / Full-Page Overprint**：按当前句词数和同墨色数量全屏连锁。

### 11. 轻度擦边边界

- **商店首屏 PG-13**：成年排字师、校对员、走私商可以有工业围裙、墨迹、束腰、手套、高跟靴等成熟造型，但不出现 nudity、性行为姿态或 adult-only 文案。
- **游戏内成熟危险感**：来自深夜工厂、墨迹沾身、压印噪音、危险交易、Boss 近距离改词的支配感。
- **擦边服务 build**：走私商的“贴身校样”“私印铅芯”等只作为高风险卡牌交易文案，实际收益是 socket、复制、禁字诅咒，不变成成人互动。

### 12. 为什么能超过红线清算局或断轨夜车

**超过红线清算局的市场稳健性：**

- 画面第一动词是“印刷机砸怪”，比签章、条款、违约更少语言门槛。
- 墨、字模、滚筒、纸带在缩略图里非常实体，不会像合同题材那样依赖 UI 文案。
- 成人危险感可通过工厂夜班和角色气场表达，审核风险低于直接性化题材。

**超过断轨夜车的设计记忆：**

- 夜车的强物件是车票，本方向的强物件是“活字版框”：它不只是资源载体，而是玩家亲手排 combo 的工作台。
- `CUT / BURN / PRESS` 这种短词在 GIF 中跨语言可读，适合短视频和直播弹幕复述。
- 句子组合天然支持 build 失控，比固定车厢推进更容易做出 Balatro 式倍增惊喜。

### 13. 最大失败风险和一句补救

**风险：** 拼字/排版可能被误解成文字解谜，拖慢杀怪节奏，偏离 Vampire Crawlers-like。

**补救：** 第一版只使用 20-30 个图标化短词，把每个词当攻击模块而不是语言谜题；所有出牌都必须在 1 秒内触发可见清怪。
