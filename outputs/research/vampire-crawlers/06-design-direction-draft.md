# 原创游戏方向草案

> **状态：已废弃，仅保留历史参考。**  
> 本文中的 `Organ Fleet / 星舰胃袋` 等早期方向不再作为当前主方案。原因见 `25-deprecated-ideas-register.md`。后续请以 `24-subagent-theme-pitch-synthesis.md` 为当前题材入口。

状态：初稿，等待 Steam 全量评测和 YouTube 评论回填后再二次修订。  
目标：从 Vampire Crawlers 的核心体验中提炼可迁移结构，并生成原创世界观下的可执行方向。

## 我对竞品核心体验的判断

Vampire Crawlers 的强点不是“第一人称地牢”本身，也不是“卡牌”本身，而是把 Vampire Survivors 的三种快感翻译到了新形式里：

1. **持续获得感**：经验、宝箱、金币、升级、解锁不断打断空白时间。
2. **构筑破局感**：玩家从小数字、小效果，一步步堆到荒谬组合。
3. **低摩擦再开局**：失败也能回村，带着货币和目标继续下一局。

它的风险也来自同一件事：如果玩家没有进入破局段，前期就会显得像按顺序出牌和刷永久升级。

所以我们的原创游戏不能只换皮。必须在第一局就交付：

- 至少两条明显可行的 build 路线。
- 一次可见的连击爆发。
- 一个失败后仍然想再开的长线目标。
- 一个与新世界观强绑定的卡牌幻想。

## 方向 A：星舰胃袋 / Organ Fleet

### 核心幻想

玩家不是勇者，而是一艘活体星舰的“器官调度员”。地牢不是地下城，而是星舰体内被寄生体感染的舱段。卡牌不是武器，而是器官脉冲、免疫反应、神经命令和代谢过载。

### 为什么适合

它天然支持 Vampire Crawlers 的核心结构：

- 第一人称推进 = 在活体舰舱中清理感染节点。
- 卡牌连击 = 神经脉冲按顺序激活器官链。
- 宝石/进化 = 器官突变、菌群共生、免疫记忆。
- 永久升级 = 星舰中枢修复、器官移植、基因库解锁。

### 核心循环

1. 从母舰中枢选择一个“驾驶人格”。
2. 进入感染舱段，按格推进。
3. 遇敌后打出器官卡牌。
4. 低耗卡启动神经链，高耗卡释放器官爆发。
5. 获得蛋白质、突变因子、免疫记忆。
6. 死亡或清理舱段后回到中枢，解锁新器官和舱段。

### 差异化突破

- 卡牌顺序不只是数字升序，而是“神经链路”：神经 -> 血液 -> 肌肉 -> 酸液 -> 免疫。
- 地牢房间可以是器官功能节点，不只是战斗格。
- UI 可以用医学/生物仪表表达资源，让世界观和系统贴合。

### 风险

- 生物题材可能偏怪，需要控制可读性。
- 卡牌关键词要避免太抽象。

### 10 分钟 demo

- 1 个感染舱段。
- 1 个驾驶人格：外科人格。
- 24 张器官卡。
- 8 种寄生敌人。
- 3 条 build：酸液、免疫、神经超频。
- 第 7 分钟出现一次“器官失控”爆发，让玩家看到后期可能性。

## 方向 B：梦境债务局 / Dream Debt Office

### 核心幻想

玩家是梦境债务局的催收员，进入欠债者的梦境迷宫，回收记忆、恐惧和未完成愿望。卡牌是合同条款、心理暗示、梦境漏洞和利息惩罚。

### 为什么适合

它把“地牢探索 + 卡牌构筑”变成了叙事更强的任务：

- 地牢 = 一个人的梦境档案。
- 敌人 = 债务情绪、防御机制、谎言化身。
- 宝箱 = 被抵押的记忆。
- 永久升级 = 债务局权限、法条、梦境装备。

### 核心循环

1. 选择一个案件。
2. 进入梦境楼层。
3. 战斗中用合同卡、催眠卡、证据卡组成连锁。
4. 通过奖励选择改变案件走向。
5. 回到债务局升级部门和条款。

### 差异化突破

- 每个地牢有轻叙事目标，比竞品“地牢只是容器”更强。
- 卡牌可以绑定“法律顺序”：取证 -> 施压 -> 结算 -> 追缴。
- 失败也可以带回案件线索，降低重复感。

### 风险

- 叙事写作量更高。
- 如果文本太多，会破坏快节奏。

### 10 分钟 demo

- 1 个债务人梦境。
- 3 层梦境楼。
- 1 个案件反转。
- 20 张卡牌。
- 5 种情绪敌人。
- 结尾给出一个“回收记忆后改变局内规则”的强反馈。

## 方向 C：废墟广播员 / Ruin Broadcast

### 核心幻想

玩家在末日废墟中驾驶移动电台，进入被静电污染的城市区块，用广播、噪声、频段、听众回声进行战斗。卡牌是频率、节目段落、广告插播、听众祈愿和噪声武器。

### 为什么适合

这个方向最适合验证“音频和战斗爽感”：

- 连击 = 节目编排顺序。
- 经验宝石 = 听众回声。
- 宝石/进化 = 频段调制器。
- 地牢 = 城市广播盲区。

### 核心循环

1. 选择主播和电台车。
2. 进入废墟区块。
3. 通过卡牌播放节目、叠频、引爆噪声。
4. 吸收听众回声升级节目单。
5. 回基地扩建电台，解锁新频段。

### 差异化突破

- 音效和节奏能成为系统的一部分，而不是包装。
- 视频分析里提到的“动态爽感”可以直接转成声画反馈设计。
- 新世界观强，和 Vampire IP 脱钩明显。

### 风险

- 对音频设计要求更高。
- 如果 Web 原型音效薄，会削弱方向优势。

### 10 分钟 demo

- 1 个废墟区块。
- 1 个主播。
- 24 张节目卡。
- 3 条 build：噪声爆破、听众召回、广告经济。
- 第 5 分钟让玩家第一次把节目段落连成“整点爆播”。

## 当前推荐

我目前推荐 **方向 A：星舰胃袋 / Organ Fleet**。

理由：

1. 它和竞品的系统骨架贴合度最高，能最大限度测试 Codex 的“洗机制”能力。
2. 它足够原创，不会靠 Vampire Survivors 的角色、武器和怪物红利。
3. 它天然支持伪 3D：活体舱段、器官房间、感染节点、神经链路都能在第一人称格子地牢里成立。
4. 它能把卡牌顺序从“数字升序”升级成“生理链路”，这是可解释的突破点。

## 需要用后续数据验证的假设

1. 玩家是否真正喜欢“顺序连击”，还是只是喜欢 Vampire Survivors 的 IP 和奖励频率。
2. 负面评论中“重复”和“慢热”出现频率有多高。
3. 视频评论区是否更在意画面爽感、build 破局、还是价格/平台。
4. 媒体批评的 UI 可读性问题，在 Steam 玩家评论里是否同样高频。
5. 便携/手柄问题是否是核心风险，还是主机媒体特有问题。
