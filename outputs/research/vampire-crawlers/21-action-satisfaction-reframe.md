# 从诊疗所改回斩杀爽感的题材修正

生成时间：2026-05-10 21:35 HKT  
触发原因：用户指出 `夜班收容：体温档案` 提案太像模拟经营，缺少《Vampire Crawlers》通过迅速斩杀怪物得到的爽感。

## 用户反馈结论

这个反馈成立。

`诊疗所/体温档案` 的问题不是题材不能擦边，而是核心动词错了：

| 原提案动词 | 问题 |
|---|---|
| 诊疗 | 像经营/视觉小说，不像战斗 |
| 记录病例 | 像图鉴收集，不像清场 |
| 复查/校准 | 反馈太轻，不够爽 |
| 关系事件 | 容易把留存重心带到角色事件 |
| 污染管理 | 容易变成数值压力，而不是爆发反馈 |

《Vampire Crawlers》的核心快感不是“系统很有设定”，而是：

1. 怪物快速涌来。
2. 玩家用卡牌/构筑迅速清掉。
3. 屏幕、音效、数字和奖励不断反馈。
4. build 从弱到强，最后出现连锁处决和局面翻盘。

所以新题材必须先满足：

**玩家能不能在 10 秒内看懂：我在一路杀怪，而且杀得越来越快。**

## 新筛选原则

以后题材先过这 5 条，不通过就淘汰：

1. **核心动词必须是杀、斩、焚、封、爆、处决、清场**，不能是诊疗、管理、收集、经营。
2. **女性吸引力必须在战斗系统里有位置**，可以是可选英雄、危险敌人、Boss、商人、异常同伴或卡牌来源，不是只在 hub 里发事件。
3. **擦边必须服务战斗反馈**，比如血誓、魔女契约、魅惑暴走、战损强化。
4. **第一屏必须有敌潮和攻击反馈**，不能只有角色立绘和 UI。
5. **10 分钟原型必须出现一次连锁清场**，否则题材再性感也不合格。

## 对旧方向的处理

`夜班收容：体温档案` 不应继续作为主方向。

可保留的元素：

- 成年女性角色。
- 异常收容。
- 污染/契约/体温等擦边文案。
- 档案作为局外解锁。

必须删除或降级的元素：

- 诊疗所作为主场景。
- 医生/病例/复诊作为核心动词。
- 关系事件作为主要留存。
- 过多 hub 叙事。

转化方式：

`夜班收容：体温档案` -> `夜班猎杀：异常清除令`

从“医生处理病例”改成“成年女猎人执行异常清除任务”。体温、污染和契约仍可存在，但都变成战斗资源。

## 修正后的 3 个方向

### A. 夜班猎杀：异常清除令

一句话：

玩家带着多名清除员进入异常设施，用规程卡、处决卡和污染契约快速斩杀成群异常。女性吸引力来自部分可选英雄、Boss、怪物潮、契约商人和异常同伴，不强制所有英雄都是女性。

为什么比诊疗所更对：

- 主动词是猎杀/清除/处决。
- 敌人是异常实体，不是“患者”。
- 女性角色进入战斗系统：清除员、处刑官、污染猎人、异常叛逃者、Boss、商人或怪物潮。
- 擦边来自战损、危险契约、污染暴走，而不是诊疗暧昧。

核心爽感：

| 时间 | 体验 |
|---|---|
| 0-2 分钟 | 手枪/刀刃/符卡清掉第一波异常 |
| 2-5 分钟 | 第一套卡牌连锁出现：标记 -> 拉近 -> 斩杀 -> 爆裂 |
| 5-8 分钟 | 污染契约给高收益，清场速度暴涨 |
| 8-10 分钟 | 第一个 Boss 或精英潮，触发一次屏幕级清场 |

擦边表达：

- 角色战损和污染纹路随 build 强化。
- 高污染卡牌有低语、心跳、红线 UI。
- 成年女性猎人台词短促、危险、压迫。
- 不做医疗检查，不做成人场景。

### B. 血月猎巫：修道院清算

一句话：

玩家是血月猎巫人，在被异化的修道院中用圣印、血刃、忏悔链和誓约卡快速处决怪潮。

为什么强：

- 哥特视觉强，商店点击强。
- 女猎人/血誓修女/忏悔处刑官都能自然擦边。
- “血月 + 清算 + 处决”比“血疗”更有动作感。
- 卡牌可直接变成战斗链：烙印、拉扯、斩首、爆血、净化。

核心爽感：

| 系统 | 表达 |
|---|---|
| 标记 | 圣印烙在敌人身上 |
| 聚怪 | 忏悔链拉成一团 |
| 斩杀 | 血刃处决低血敌人 |
| 爆发 | 血月共鸣造成连锁爆血 |
| 长线成长 | 猎巫执照、圣器改造、禁忌誓约 |

擦边表达：

- 成年女猎人和血誓修女，强调战斗和禁忌。
- 声音是钟声、皮革、链条、低声忏悔，不是露骨声音。
- 文案是誓约、忏悔、圣痕、处决许可，不是性描写。

风险：

- 宗教/修女擦边容易俗套，需要架空宗教。
- 血腥和擦边一起上，审核风险高于 A。

### C. 魔女处刑局：禁忌合同

一句话：

玩家是魔女处刑局执行人，在合同迷宫中用咒印、契约漏洞和处刑卡清除违约恶魔。

为什么强：

- 保留魔女/女性角色/契约擦边，但核心是处刑和清场。
- 合同、条款、违约、印章都能变成卡牌机制。
- 比魅魔题材平台风险低，比诊疗所更有战斗感。
- UI 和文案可以非常有识别度。

核心爽感：

| 系统 | 表达 |
|---|---|
| 合同标记 | 给敌人挂违约条款 |
| 处刑触发 | 满足条款后自动斩杀 |
| 印章爆发 | 一次盖章引爆整排敌人 |
| 契约漏洞 | 高收益卡，但给自己加债务/诅咒 |
| Boss | 大恶魔客户拒绝履约，进入强制清算 |

擦边表达：

- 成年契约魔女、恶魔客户经理、处刑律师。
- 声音是钢笔、印章、低语、合同燃烧。
- 文案是“签署”“违约”“第七条代价”“清算开始”。
- 性感来自权力、危险交易和职业姿态，不靠露骨。

## 当前重新排序

| 排名 | 方向 | 原因 |
|---:|---|---|
| 1 | 血月猎巫：修道院清算 | 最强视觉点击和斩杀爽感，但风险略高 |
| 2 | 夜班猎杀：异常清除令 | 最均衡，保留异常收容差异化，同时回到清场 |
| 3 | 魔女处刑局：禁忌合同 | 最稳，文案和系统强，但画面爽感要加强 |

如果目标是“100 个游戏里脱颖而出”，我现在不再推荐诊疗所。更推荐：

**血月猎巫：修道院清算**

因为它第一眼就有更强的动作承诺：血月、猎巫、修道院、女猎人、处决、怪潮。它天然比“诊疗所”更接近《Vampire Crawlers》的杀怪爽感。

## 原型验证要求

无论选哪个方向，第一版原型必须包含：

1. 每 3-5 秒一次小清场。
2. 每 30-60 秒一次 build 反馈。
3. 第 5-8 分钟一次大范围连锁处决。
4. 至少 3 种敌人死亡反馈：爆裂、燃烧、封印、斩断、溶解。
5. 角色擦边反馈只做战斗强化：战损、污染、契约暴走，不做模拟经营事件。

## 对后续 GDD 的改写要求

正式 GDD 必须先写战斗：

1. 玩家如何移动。
2. 怪物如何涌来。
3. 卡牌如何杀怪。
4. build 如何提高清场效率。
5. 女性角色和擦边如何增强战斗动机。

不能先写 hub、关系、档案、角色事件。
