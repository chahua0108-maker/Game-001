# 前三小时体验理解缺口

> 2026-05-10 20:10 HKT 更新：本文件记录的是早一阶段的理解边界。后续已补充三条长 VOD 的材料，详见 `13-first-3-hours-video-map.md` 和 `14-first-3-hours-experience-analysis.md`。当前状态已经从“不能深刻理解前三小时”推进到“可以提炼前三小时核心体验曲线，但仍有少量画面证据缺口”。

更新时间：2026-05-10

## 结论

当前还不能说已经“深刻感受到前三个小时的核心体验”。

我目前理解到的是：

- 首局进入方式。
- 第一人称地牢移动和遭遇切换。
- 卡牌 UI 的基本结构。
- 连击/出牌/奖励的短片段表现。
- 媒体评测对节奏、重复、爽感、UI 可读性的分歧。
- 部分 20 分钟内 build 成型和爆发片段。

但这还不足以判断前三小时体验，因为前三小时的关键不在“战斗长什么样”，而在体验曲线：

1. 第 0-15 分钟：玩家是否快速理解核心循环。
2. 第 15-45 分钟：第一次 build 是否成型。
3. 第 45-90 分钟：永久升级是否开始驱动下一局。
4. 第 90-180 分钟：重复感、刷资源、UI 摩擦、破局爽感哪个开始占主导。

## 当前已有视频证据

| 类型 | 已有材料 | 能说明什么 | 不能说明什么 |
|---|---|---|---|
| 官方 trailer | announcement / gameplay / launch | 官方希望卖什么、画面和 UI 基本形态 | 真实节奏、重复、失败成本 |
| GameSpot 评测 | 1/2/4 分钟附近短片段 | 媒体如何讲地牢、战斗、构筑 | 连续游玩时的疲劳点 |
| Nintendo Life / SwitchUp | 掌机/战斗/升级片段 | 便携设备可读性、UI 和性能口径 | PC 上 3 小时体验 |
| The Spiffing Brit | 5/10/20 分钟短片段 | 开荒和破局的部分动态表现 | 1-3 小时成长曲线 |

## 需要补的证据

### 长视频 / VOD

需要至少 2-3 条 60 分钟以上视频，最好有一条覆盖 3 小时。

观察时间码：

- `0:00-0:15`：新手、第一战、第一次升级。
- `0:15-0:45`：第一条 build 路线是否出现。
- `0:45-1:30`：重复感和永久升级是否开始显著。
- `1:30-3:00`：破局、后期目标、刷资源、失败/通关后循环。

### 评论区

重点不看普通“好玩/不好玩”，只看和时间曲线有关的评论：

- 玩了多久开始上头。
- 玩了多久开始觉得重复。
- 是否因为永久升级或金币而继续。
- 是否觉得 UI/卡牌排序在长局里变累。
- 是否认为前作 IP 支撑了耐心。

### 字幕/口播

评测者常会直接说：

- “after X hours...”
- “once you unlock...”
- “the early game...”
- “the grind...”
- “the loop opens up...”

这些比 trailer 更能说明前三小时体验。

## 分析原则

在补齐长线证据前，不做以下判断：

- 不断言“前三小时很好玩”。
- 不断言“重复问题严重”。
- 不把 20 分钟片段当成完整成长曲线。
- 不把官方 trailer 当玩家真实体验。

当前可以做的判断：

1. 核心流程是可被 Web 原型表达的。
2. 爽感来自连续反馈和 build 破局，不是单张截图。
3. 最大风险很可能在 45-180 分钟区间，而不是前 10 分钟。
4. 我们的原型如果只做 10 分钟，必须额外模拟一次“中后期爆发”，否则无法验证真正卖点。
