# Prototype GDD：Organ Fleet / 星舰胃袋

> **状态：已废弃，不进入原型开发。**  
> `Organ Fleet / 星舰胃袋` 与当前“轻度擦边 + 成年女性角色 + 快速清怪 + Steam 首屏突围”的验证目标不匹配。本文仅保留为早期 GDD 样例和机制素材，当前 GDD 请从 `24-subagent-theme-pitch-synthesis.md` 和 `25-deprecated-ideas-register.md` 进入。

状态：草案 v0.1  
依据：当前媒体证据、Steam 近期大样本、YouTube 动态材料。  
目标：做一个 10 分钟 Web 原型，验证 Codex 能否把竞品核心流程洗成原创世界观与可迁移设计。

## 一句话

玩家是活体星舰的器官调度员，进入被寄生体感染的舱段，通过神经链路式卡牌连击激活器官武器，清理感染、收集突变、修复星舰中枢。

## 体验目标

1. 第一局 2 分钟内理解：移动、遭遇、出牌、连击、升级。
2. 第 5 分钟内感到：我正在形成一个 build。
3. 第 7-9 分钟出现一次明显爆发：多张器官卡连锁导致敌群清屏。
4. 死亡或通关后明确看到：下一局可以通过中枢修复变强。

## 玩家身份

玩家不是传统战士，而是“调度人格”。每个调度人格代表星舰中枢的一种偏执生存策略。

首版 3 个角色：

1. 外科人格
   - 擅长单体切除、流血、精确爆发。
   - 初始卡偏攻击。

2. 免疫人格
   - 擅长护盾、反击、感染净化。
   - 初始卡偏防御和反击。

3. 代谢人格
   - 擅长资源转化、抽牌、过载。
   - 初始卡偏经济和连锁。

## 核心循环

1. 在中枢选择人格和舱段。
2. 进入第一人称格子舱段。
3. 选择路径：感染节点、资源节点、器官节点、休眠舱、Boss 门。
4. 遇敌进入卡牌战斗。
5. 打出神经链路：
   - 神经 0 费
   - 血液 1 费
   - 肌肉 2 费
   - 酸液 3 费
   - 免疫 Wild
6. 杀敌获得蛋白质和突变经验。
7. 升级选择新卡、卡牌插槽、器官突变或中枢资源。
8. 清理楼层 Boss，继续下一层或回中枢。
9. 失败后保留部分资源，解锁新器官和永久修复。

## 战斗系统

### 基础资源

- `impulse`：每回合行动资源，对应 mana。
- `heat`：过载风险，使用强卡会累积。
- `tissue`：临时护甲和自修复。
- `mutation_xp`：局内升级进度。

### 连击规则

竞品是按费用升序触发强化。Organ Fleet 改成“生理链路”：

1. 神经卡启动链路。
2. 血液卡放大后续资源和流动效果。
3. 肌肉卡放大伤害和位移。
4. 酸液卡造成爆发/群体腐蚀。
5. 免疫 Wild 可以重置链路或复制上一个链路节点。

这样保留“顺序连击”的核心体验，但解释权来自原创世界观。

### 第一版卡牌池

每类 6 张，共 30 张：

- 神经：触发、抽牌、减费、预判。
- 血液：回血、加速、资源转化、扩散。
- 肌肉：斩击、冲击、护甲、击退。
- 酸液：腐蚀、溅射、穿透、爆炸。
- 免疫：净化、复制、反击、Wild 链路。

## 地牢系统

### 空间表现

- Three.js 第一人称格子移动。
- 每格代表一个舱段节点。
- 敌人以 billboard sprite 或低模显示。
- 战斗时镜头锁定敌群，UI 覆盖下方卡牌区。

### 节点类型

- 感染节点：普通战斗。
- 囊肿节点：精英战斗。
- 蛋白质池：金币/资源。
- 器官台：卡牌插槽或进化。
- 休眠舱：回血或移除污染。
- 神经门：Boss 前置门。

## 成长系统

### 局内

- 升级给 3 选 1。
- 选项类型：
  - 新卡。
  - 卡牌插槽。
  - 器官突变。
  - 链路关键词增强。

### 局外

- 中枢修复树：
  - 初始手牌 +1。
  - 每局首个器官台必出进化。
  - 解锁新人格。
  - 解锁新舱段。
  - 解锁新卡类。

## 10 分钟 Demo 流程

| 时间 | 玩家体验 | 系统要求 |
|---:|---|---|
| 0:00-0:30 | 中枢启动，选择外科人格 | 主菜单 + 角色选择 |
| 0:30-1:30 | 第一段移动和第一个感染节点 | 伪 3D 移动 + 战斗切入 |
| 1:30-2:30 | 第一次神经 -> 血液 -> 肌肉连击 | 卡牌 UI + 连击预览 |
| 2:30-4:00 | 升级，选择酸液卡或免疫 Wild | 奖励选择 |
| 4:00-6:00 | 第二层，出现精英敌人 | 地牢路径 + 精英 |
| 6:00-8:00 | build 成型，第一次群体爆发 | 反馈、粒子、音效、数字 |
| 8:00-10:00 | Boss 或强制死亡，回中枢 | 结算 + 永久升级入口 |

## UI 要求

必须优先解决竞品被批评的问题：

1. 卡牌上显示链路类型和下一个推荐链路。
2. 出牌前预览连击倍率。
3. 奖励选择显示“这会增强哪条路线”。
4. 爆发时保留主要数字和状态可读。
5. 手柄/键盘输入映射明确。

## 不做内容

首版不做：

- 大量剧情文本。
- 复杂 3D 模型。
- 多地图大世界。
- 在线功能。
- 商业化 Steam 包体。

## 成功标准

1. 试玩者能用自己的话说出“我在激活器官链路”。
2. 试玩者能在第一局看到一次爆发。
3. 试玩者失败后知道下一局可以怎么变强。
4. 数据结构不依赖 Three.js，可以迁移到 Unity/UE。
5. 所有资产和文本都是原创或占位，不触碰竞品资产。
