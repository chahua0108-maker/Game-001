# Subagent C 题材提案：轻度擦边、审核稳健、市场突围

生成时间：2026-05-10 HKT  
角色：题材提案 subagent C  
目标：提出 2 个原创 Vampire Crawlers-like 题材方向，重点解决“轻度擦边如何提高点击、留存和角色记忆，但不把项目导向 Adult Only、黄油标签或慢节奏关系事件”。

## 共同判断

这两个方向都不把“魅魔”“美少女小队”“诊疗所”“温泉旅馆”当主语。它们的第一屏必须是：

`走廊/车厢压迫 -> 怪潮涌来 -> 出牌清怪 -> 连锁爆发 -> 风险资源上涨 -> 下一扇门`

轻度擦边只放在成年角色气质、危险契约、短台词、污染状态、Boss 压迫、局外解锁里。不能做露骨成人 CG、慢节奏关系事件、检查/护理/温泉互动、可付费亲密内容或隐藏成人素材。

平台基线以本地 `18-risk-review-landscape.md` 为准，并补充核对了 Steamworks 当前公开文档：Content Survey 要求如实披露 mature content 和 AI 内容，Graphical Asset Rules 要求 capsule PG-13，Community Items 要求全球全年龄，Onboarding 规则提到支付处理商/卡组织/银行标准，PayPal AUP 对 sexually oriented materials/services 保持敏感。因此，本文件所有提案默认走“成熟暗示 + 动作清场”，不走 Adult Only Sexual Content。

## 提案 1：红线清算局：心跳回收 / Redline Repossession: Heartbeat Recovery

### 1. 一句话 hook

一支超自然清算队进入不断延展的“欠债走廊”，用红线、签章、心跳回收和违约处刑卡，从吸血鬼债务人和异常债主手中抢回被抵押的生命脉冲。

### 2. 为什么它能在 100 个同类里跳出来

市场上常见的是魔女、修女、魅魔、女猎人、地牢和吸血鬼。`红线清算局` 的新鲜点是把“危险亲密感”从色情联想转成“心跳被抵押、红线缠腕、近距离签押、债务反噬”。玩家第一眼看到的是红线切割怪潮、印章爆裂、心跳 UI 和职业感很强的成年清算员，不是成人角色池。

这个壳天然适合卡牌：条款、签章、违约、回收、质押、清算都可以直接变成关键词。轻度擦边来自成年女性债权人、黑票经纪人、红线执达官的压迫感和短台词，而不是福利事件。

### 3. 成年女性角色 / 女性怪物 / Boss / 商人 / 异常同伴分布

| 位置 | 分布结构 | 功能 |
|---|---|---|
| 可选英雄 | 男女/中性清算员混合；其中 2 名成年女性职业入口：红线执达官、脉冲审计师 | 分别对应处刑流、抽牌流 |
| 女性怪物 | 约 25%-35% 怪潮为女性轮廓异常，如赊血秘书、镜债舞者、红线寄生者 | 提供魅惑、拉扯、手牌污染、聚怪词缀 |
| 女性 Boss | 逾期夫人、第一债权人、心跳公证员 | 每个 Boss 都给一套高风险契约卡 |
| 女性商人 | 黑票经纪人，成年、冷静、职业化 | 出售诅咒牌、回收牌、代价牌 |
| 异常同伴 | 缠腕红线、抵押心脏、无声书记 | 击败 Boss 后可转化为召唤物、遗物或局外成长节点 |

### 4. 10 秒杀怪承诺

第一扇门打开，走廊尽头涌来一批“逾期者”。玩家先打出 `违约盖印` 给前排挂标记，再打 `红线收束` 把两侧怪拉到中线，最后打 `心跳回收` 处决低血目标。被处决的敌人爆成红色签章和脉冲碎片，连锁引爆身后的标记怪。10 秒内玩家明确看到：这不是关系事件，这是一路清算、切割、爆怪。

### 5. 10 分钟原型核心循环

| 时间 | 玩家体验 | 原型必须验证 |
|---|---|---|
| 0-1 分钟 | 选择清算员，进入欠债走廊，第一波怪潮压近 | 开局 30 秒内杀怪 |
| 1-3 分钟 | 学会 `盖印 -> 收束 -> 回收` 的基础链 | 3 张卡形成一次小清场 |
| 3-5 分钟 | 黑票经纪人出现，给出高收益但加债务的牌 | risk/reward 不是文本，而是战斗数值 |
| 5-7 分钟 | 债务层数升高，怪物更快逼近，但清场收益翻倍 | 污染/债务条和出牌节奏绑定 |
| 7-9 分钟 | 小 Boss `逾期夫人` 在走廊尽头推进 | Boss 压迫和怪潮同时存在 |
| 9-10 分钟 | 触发 `最终清算`，全屏红线切割，获得局外心跳碎片 | 必须有一次 build 爆发和局外收益 |

### 6. 擦边元素如何转成机制

| 擦边元素 | 转化方式 | 风险控制 |
|---|---|---|
| 红线缠腕 | `红线污染` 状态：提高暴击和抽牌，但满层会让敌人贴近速度上升 | 视觉是手腕符线和 UI，不做束缚成人化 |
| 近身签押 | 敌人距离越近，签押卡伤害越高；失败则加 `欠息` 诅咒 | 表达为职业动作和战斗危险，不写亲密挑逗 |
| 心跳回收 | 处决低血敌人后回血或回费 | 心跳是生命资源，不做身体检查 |
| 黑票交易 | 商人给强卡，同时塞入债务牌 | 商人对话短促，最多一句压迫台词 |
| 局外解锁 | 解锁签章、语音短句、Boss 档案、卡背 | 不做成人图库，不把服装露出当奖励 |

短台词样例：

- 红线执达官：“你的心跳，逾期了。”
- 黑票经纪人：“签下去，这一局你会赢。下一局再还。”
- 逾期夫人：“靠近一点，我才能确认你的利息。”

首屏画面：一名成年清算员半侧身站在透视走廊前，手腕红线延伸成切割轨迹，远处怪潮压来，前景有 3 张发光卡牌和心跳 UI。女性角色可以有成熟、冷峻、危险的职业魅力，但构图重心必须是红线切怪、签章爆裂、怪潮和卡牌。

### 9. Steam / 支付 / 社区物品 / AI 内容风险规避

- Steam Content Survey：如实披露风格化暴力、恐怖、成熟暗示；不要上传隐藏成人素材。

### 10. 为什么比直接魅魔 / 直接美少女 / 诊疗所 / 温泉旅馆更稳也更能卖

- 比直接魅魔稳：不把玩家期待导向 Adult Only，也不吸引“必须有成人内容”的评价压力。
- 比直接美少女强：角色记忆点来自红线、签章、心跳、债务和清算职业，不是普通角色池。
- 比诊疗所快：核心动词是切割、盖印、处决、爆裂，不会滑成检查、档案、关系事件。
- 比温泉旅馆更适合 Vampire Crawlers-like：走廊、债务怪潮、红线切割天然服务 10 秒清怪和 10 分钟 build 爆发。
- 更能卖的点：标题和首屏有独特识别，轻度擦边被包装成危险职业魅力和机制代价，既能提高点击，也不会牺牲平台可见性。

## 提案 2：暗香猎列：第九车厢 / Nocturne Express: Scent of the Ninth Car

### 1. 一句话 hook

玩家登上一列被吸血贵族污染的午夜猎列，用银雾、折扇风切、香轨爆燃和车厢封锁卡，在一节节狭窄车厢里清除被“夜香”吸引来的怪潮。

### 2. 为什么它能在 100 个同类里跳出来

`暗香猎列` 把轻度擦边从身体暴露转成“气味、面具、车厢、成人晚宴、危险靠近”。它不是温泉、旅馆、诊疗或魅魔，而是一个能直接服务伪 3D 走廊压迫的场景：火车车厢天然狭窄，门一开就是下一波怪，窗外高速移动，Boss 可以从远处车厢一步步逼近。

“香水/夜香”提供了非常强的 risk/reward 资源：香气越浓，卡牌越强、怪物越容易被引诱聚拢，但玩家也更容易手牌污染、视野变窄、Boss 提前出现。性感来自成年香术师、列车女公爵、假面乘务长、危险低语和华丽战斗姿态，不来自成人事件。

### 3. 成年女性角色 / 女性怪物 / Boss / 商人 / 异常同伴分布

| 位置 | 分布结构 | 功能 |
|---|---|---|
| 可选英雄 | 不锁女性主角；职业含猎列枪手、银雾调香师、车厢驱魔人。其中银雾调香师为成年女性高风险法术入口 | 对应雾化、引燃、聚怪、控场 |
| 女性怪物 | 香影侍女、假面舞客、玫瑰寄生体等，占怪潮一部分 | 释放香雾、魅惑手牌、召唤幻影 |
| 女性 Boss | 第九车厢女公爵、黑玫瑰乘务长、无脸歌姬 | 每个 Boss 控制一种香调和车厢规则 |
| 女性商人 | 断票调香师，成年、戴手套、职业化 | 出售香调卡、换气卡、污染转化卡 |
| 异常同伴 | 活香瓶、折扇影、银雾猎犬 | 击败 Boss 后变成召唤物或局外升级 |

### 4. 10 秒杀怪承诺

车厢门被撞开，香影怪从过道两侧挤入。玩家打出 `银雾喷瓶` 在前方形成锥形雾区，再用 `折扇风切` 把雾区中的敌人切成两段伤害，最后 `香轨爆燃` 点燃地面残留香迹，火线沿车厢一路传到门口，连续炸死后排怪。10 秒内玩家看到的是狭窄车厢里大面积清场、怪物爆成玻璃香片、卡牌连锁成立。

### 5. 10 分钟原型核心循环

| 时间 | 玩家体验 | 原型必须验证 |
|---|---|---|
| 0-1 分钟 | 从第 1 车厢开始，学会银雾和风切 | 火车车厢作为走廊压迫成立 |
| 1-3 分钟 | 怪潮被香气吸引聚拢，玩家用雾区清场 | 香气不是装饰，而是聚怪/伤害系统 |
