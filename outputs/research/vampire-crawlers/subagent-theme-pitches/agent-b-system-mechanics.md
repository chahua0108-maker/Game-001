## 提案 A：断轨夜车：零号车厢 / Nightrail Culling: Car Zero

### 1. 暂定标题

中文：**断轨夜车：零号车厢**  
英文：**Nightrail Culling: Car Zero**

### 2. 一句话 hook

玩家在一列永不到站的夜车中执行清场令，用检票、打孔、改轨、煤炉、断钩卡把逼近的乘客怪潮一节节清空，最后把整段车厢脱轨成一次屏幕级处决。

### 3. 题材到系统的直接映射

| 系统 | 题材映射 | 战斗作用 |
|---|---|---|
| 抽牌 | 每回合翻开“时刻表”，抽到的卡就是本节车厢可执行的车务命令 | 抽牌不是菜单奖励，而是夜车运行节奏 |
| 费用 | `煤压`，打牌消耗煤压；牺牲卡可把票据丢进煤炉换煤压 | 让高爆发和过热风险同源 |
| 颜色 | 红线、蓝线、金线、黑线四种车票颜色 | 正确颜色连锁；错色可强打但涨“脱轨风险” |
| Combo | `检票 -> 打孔 -> 改轨 -> 引燃 -> 断钩` | 从单体标记变成聚怪、燃烧、处决、清场 |
| Upgrade | 车票升舱：普通票、月台票、零号票 | 升级改变触发条件和清场范围 |
| Socket | 打孔留下的孔位就是 socket | 每张牌最多 1-3 个孔位，可塞入铆钉、煤晶、红铜票钉 |
| Gem | `煤晶 / 铜钉 / 骨白票钉` | 改变伤害形态、抽牌、连锁重复次数 |
| 牺牲 | 把一张正确颜色车票投入煤炉 | 立刻回煤压并触发颜色死亡反馈 |
| Reroll | `改签`，重排本节车厢奖励和下一手牌 | 视觉上是票面被盖章改线，代价是多刷一波怪 |

差异化点：卡牌不是抽象图标，而是可被打孔、改签、烧毁、升舱的实体车票。玩家每次强化都能在卡面上看到孔位和铆钉，系统反馈天然可视化。

### 4. 10 秒杀怪承诺

开局就是一条窄车厢。第 1 秒抽到 `检票`，远端三列乘客怪被红色票夹标记；第 3 秒 `打孔` 在怪物胸口打出发光孔；第 5 秒 `红线改轨` 把三列怪拉到同一轨道；第 7 秒 `餐车引燃` 沿孔位点燃；第 10 秒 `断钩` 切开车厢，前方一整排怪被甩出画面，掉落煤晶和新票。

### 5. 10 分钟原型核心循环

| 时间 | 循环内容 |
|---|---|
| 0-1 分钟 | 玩家在第一节硬座车厢内前进，学习 `检票`、`打孔`、`手枪补刀`。小怪 2-3 秒死亡一次。 |
| 1-3 分钟 | 出现三色车票。玩家学会同色连锁：红线燃烧、蓝线冻结、金线掉钱、黑线处决。 |
| 3-5 分钟 | 第一位车厢商人出现，给一张带 socket 的票。玩家选择煤晶或铜钉，形成第一条 build 路线。 |
| 5-6 分钟 | 夜车进入长隧道，怪潮密度翻倍，煤压开始不足。系统发出“投入一张正确颜色票可保命”的牺牲教学。 |
| 6-8 分钟 | 大爆发窗口：玩家用 `检票 -> 打孔 -> 红线改轨 -> 零号车厢 -> 断钩` 形成三色打孔链。零号车厢复制上一张清场牌，整节车厢脱轨，屏幕前方怪潮被一次性甩空。 |
| 8-10 分钟 | 精英 `煤炉司炉官` 接管车头，强制升温。玩家靠已有 build 保持清场，击败后解锁“煤炉 socket”作为局外成长预告。 |

### 6. 3 个成年女性角色/敌人/商人/同伴位置

| 位置 | 角色 | 如何进入战斗或发卡机制 |
|---|---|---|
| 可选英雄 | **赤票检车长 赫斯塔**，成年女性清除员 | 初始牌组偏 `检票/打孔`。她的被动是每第 3 次打孔抽 1 张牌，把女性角色直接变成抽牌和 combo 引擎。 |
| 商人 | **红绒餐车长 薇兰**，成年女性黑市商人 | 在餐车节点出现，售卖高收益 `宴席票`。这些牌能群体引燃，但会加入 `饥饿乘客` 诅咒卡，放大 risk/reward。 |
| 精英/同伴 | **煤炉司炉官 玛洛**，成年女性敌人，击败后可成为煤炉同伴 | Boss 战中她提高煤压上限并召唤炉渣怪。击败后给 `煤炉 socket`，允许玩家牺牲正确颜色牌触发一次免费小清场。 |

### 7. 10 张核心卡牌/关键词，按 combo 链排列

1. **检票 / Ticket Check**：给最前排敌人挂 `未验票`，若目标死亡则抽 1 张。
2. **打孔 / Punch Mark**：把 `未验票` 变成 `孔位`，孔位敌人受到连锁效果。
3. **红线改轨 / Redline Switch**：把带孔位的敌人拉到一条轨道，形成聚怪。
4. **煤压升温 / Boiler Pressure**：获得 2 点煤压；若本回合已打孔，额外获得 1 点煤压但涨过热。
5. **餐车引燃 / Dining Spark**：点燃所有孔位敌人；同色车票每多一张，燃烧跳跃一次。
6. **蓝票冻结 / Blue Ticket Clamp**：冻结逼近速度最快的一列怪，给玩家留出出牌窗口。
7. **金线验资 / Goldline Claim**：处决低血孔位敌人并掉落钱票，服务中期商店。
8. **零号车厢 / Car Zero**：若场上有 3 个颜色孔位，复制上一张攻击牌。
9. **断钩 / Decouple**：牺牲一张正确颜色票，处决本节车厢所有低血敌人。
10. **终点脱轨 / Night Terminus**：本局首次三色连锁后可触发，整条走廊清屏并重置过热。

### 8. 伪 3D/走廊表现方案

不做高成本 3D 车厢。画面采用固定第一人称走廊：

- 车顶灯、座椅、窗框用 2D 分层透视贴图，按距离缩放和横向 parallax。
- 敌人是 2D billboard，从远端门框按 z-depth 放大逼近。
- 每节车厢用同一套模块换色：硬座、餐车、卧铺、煤炉车只替换侧墙贴图和灯色。
- 清场反馈集中在前方透视线：红线切割、票孔火花、断钩时整段车厢向后甩出。
- UI 中卡牌真的显示孔位、铆钉和烧焦边，让 socket/gem 强化一眼可读。

### 9. 擦边表达边界

- 视觉：成年女性制服、手套、战损外套、煤灰、压迫姿态；不做内衣、露点、胸臀中心裁切或走光奖励。
- 声音：列车广播、近距离耳语、皮手套打孔、煤炉心跳、急促呼吸只作为战斗压力；不用成人呻吟或性行为音效。
- 文案：使用“检票、清场、超载、改签、断钩、下一站无人生还”等战斗化词汇；不写性行为承诺，不做视觉小说亲密事件。
- 奖励：角色语音、战斗姿态、卡牌皮肤必须绑定能力或 build，不做图库奖励。

### 10. 最大失败风险与补救

最大风险：铁路/车票机制太新奇，玩家把它理解成解谜或桌面系统，而不是快速清怪动作游戏。  
补救：

- 第一版原型前 30 秒只出现 5 张直接杀怪牌，不开放复杂改签。
- 每张核心牌都必须产生明显命中、聚怪、爆燃或处决反馈。
- `打孔` 只保留一个可视状态，不叠太多票务词。
- 大爆发的目标不是算对规则，而是让玩家看到一节车厢被整段甩空。

## 提案 B：黑潮灯塔：棱镜清场令 / Blacktide Beacon: Prism Culling

### 1. 暂定标题

中文：**黑潮灯塔：棱镜清场令**  
英文：**Blacktide Beacon: Prism Culling**

### 2. 一句话 hook

玩家在被黑潮吞没的灯塔中爬升，用点灯、雾号、棱镜、反射镜和白光合束卡把从螺旋楼梯涌上的海怪、镜影和沉歌女妖扫成灰。

### 3. 题材到系统的直接映射

| 系统 | 题材映射 | 战斗作用 |
|---|---|---|
| 抽牌 | `灯鼓` 每转一格抽一张，手牌就是当前灯塔可装入的镜片和信号 | 抽牌和灯塔旋转同步，强化节奏感 |
| 费用 | `灯油 / 电荷` | 玩家必须在持续照射和爆发扫光之间取舍 |
| 颜色 | 红、蓝、绿、紫、白五种光 | 红灼烧、蓝减速、绿诱聚、紫封印、白处决 |
| Combo | `点灯 -> 雾号 -> 诱光聚怪 -> 棱镜折射 -> 白光合束` | 颜色不是装饰，而是清场链条 |
| Upgrade | 镜片抛光、灯芯加粗、灯鼓扩容 | 升级改变光束宽度、持续时间和穿透 |
| Socket | 镜片边缘有 1-3 个 prism socket | 让同一张卡可装不同棱镜碎片 |
| Gem | 红盐晶、蓝潮晶、绿磷晶、紫封晶 | 改变光束形态，支持 build 分化 |
| 牺牲 | 烧掉一张镜片牌补灯油 | 牺牲正确颜色可立刻触发对应死亡反馈 |
| Reroll | `换灯鼓`，旋转灯鼓重抽颜色排列 | 代价是黑潮上涨，怪物推进一格 |

差异化点：卡牌颜色和走廊表现完全同构。玩家不是在 UI 里凑颜色，而是在螺旋楼梯上真的转动一束光，颜色组合直接改变屏幕中清怪轨迹。

### 4. 10 秒杀怪承诺

第 0 秒玩家点亮灯塔，前方螺旋楼梯有黑潮怪爬上来；第 2 秒 `雾号` 把它们震停；第 4 秒 `绿诱光` 把两侧敌人拉进同一光带；第 6 秒 `红棱镜` 沿楼梯烧出一条弧线；第 8 秒 `反射镜面` 把红光折回；第 10 秒第一波敌人被双弧光清空，屏幕掉落棱镜碎片。

### 5. 10 分钟原型核心循环

| 时间 | 循环内容 |
|---|---|
| 0-1 分钟 | 玩家在灯塔底层学习 `点灯` 和 `雾号`。小怪从楼梯下方爬升，第一张攻击牌就能扫死一排。 |
| 1-3 分钟 | 红、蓝、绿三色出现。红负责击杀，蓝负责减速，绿负责聚怪，玩家自然学会颜色职责。 |
| 3-5 分钟 | 棱镜走私商出现，提供第一个 socket/gem。玩家选择宽光束、连锁弹射或击杀回灯油。 |
| 5-6 分钟 | 黑潮上涨，敌人逼近速度翻倍。系统给出 `换灯鼓` 和 `烧镜补油`，让玩家在 reroll 与牺牲之间做风险选择。 |
| 6-8 分钟 | 大爆发窗口：玩家打出 `绿诱光 -> 红棱镜 -> 蓝潮透镜 -> 反射镜面 -> 白光合束`。三色对齐后变成白光，沿螺旋楼梯扫一整圈，直接清空屏幕并击碎精英护盾。 |
| 8-10 分钟 | 登上中层灯室，遇到 `沉歌主母` 的镜影 Boss。她会唱反向灯鼓，让玩家必须用已有 build 抵抗手牌污染。击败后解锁紫光封印路线。 |

### 6. 3 个成年女性角色/敌人/商人/同伴位置

| 位置 | 角色 | 如何进入战斗或发卡机制 |
|---|---|---|
| 可选英雄 | **守灯执行官 薇拉**，成年女性猎手 | 初始牌组偏红/蓝，能力是每次白光合束后回收一张已牺牲镜片，直接服务 build 翻盘。 |
| 商人 | **棱镜走私商 洛娜**，成年女性商人 | 在安全灯室出现，售卖 prism gem 和带 socket 的镜片。她的交易会降低灯油上限，换来更强清场。 |
| Boss/卡源 | **沉歌主母 纳娅**，成年女性海妖 Boss | 战斗中以歌声污染手牌，加入 `潮声` 诅咒。击败后她不是恋爱奖励，而是变成一组高风险 `沉歌` 卡，可聚怪但会加速黑潮。 |

### 7. 10 张核心卡牌/关键词，按 combo 链排列

1. **点灯 / Ignite Wick**：获得基础光束，造成小扇形伤害。
2. **雾号 / Fog Horn**：震退或减速最近一圈敌人，制造出牌窗口。
3. **绿诱光 / Green Lure**：把两侧敌人拉入当前光束轨道。
4. **红棱镜 / Red Prism**：对轨道内敌人造成灼烧，击杀后溅射红光。
5. **蓝潮透镜 / Blue Tide Lens**：减速并让灼烧持续时间翻倍。
6. **紫封标 / Violet Seal**：给精英挂封印，封印破裂时触发小范围处决。
7. **反射镜面 / Reflector Plate**：复制上一道颜色光束，但方向反转。
8. **换灯鼓 / Rotate Drum**：reroll 当前颜色排列并抽 1 张，黑潮推进一格。
9. **白光合束 / White Convergence**：若本回合打出 3 种颜色，合成白光处决低血敌人。
10. **塔顶扫海 / Beacon Sweep**：白光合束后可升级获得，沿整条螺旋楼梯清屏。

### 8. 伪 3D/走廊表现方案

不做完整 3D 灯塔。画面采用“螺旋走廊伪 3D”：

- 中央灯塔柱和楼梯扶手用 2D 弧形分层贴图，按深度缩放。
- 敌人是 2D 剪影 billboard，沿预设螺旋路径移动，距离越近越大。
- 光束是屏幕空间扇形 mask，穿过敌人时触发烧灼、冻结、封印等粒子。
- 每层灯室只替换背景色和前景遮罩：底层盐雾、中层镜室、上层风暴。
- 白光合束时不需要复杂模型，只用全屏旋转光带、敌人剪影碎裂和掉落棱镜碎片完成清场爽感。

### 9. 擦边表达边界

- 视觉：成年女性守灯人、商人和海妖采用雨衣、长靴、潜水面罩、湿发剪影、灯光边缘轮廓；不做暴露服装、胸臀裁切或被俘姿势。
- 声音：海妖歌声、无线电低语、风暴、灯油燃烧、面罩呼吸都服务压迫和怪潮预警；不用露骨语音。
- 文案：可以写“潮声贴近耳后”“别让她的歌声替你转动灯鼓”，但必须指向手牌污染、聚怪或风险收益，不写成人行为。
- 奖励：沉歌卡、棱镜皮肤、角色短句都必须改变战斗策略，不做 adult-only、视觉小说或图库奖励。

### 10. 最大失败风险与补救

最大风险：颜色和棱镜系统可能被玩成抽象三消，杀怪爽感被颜色规则盖住。  
补救：

- 每种颜色第一张牌都必须能直接影响敌人死亡或逼近速度。
- UI 只显示当前三格灯鼓，不展示复杂色轮。
- 颜色 combo 的最终反馈统一落到 `白光合束`，让玩家记住“凑色是为了清屏”。
- 怪物必须持续从螺旋楼梯压近，避免玩家只盯手牌不看走廊。

## 两个提案的共同原型红线

1. 开局 30 秒必须杀怪，不能先讲设定。
2. 每 3-5 秒有小清场，每 30-60 秒有 build 反馈。
3. 第 5-8 分钟必须出现一次玩家可主动打出的屏幕级清场。
4. 成年女性角色必须是英雄、Boss、商人、同伴或卡源，不能只是 hub 立绘。
5. 轻度擦边只服务战斗张力、风险交易和 build 选择，不进入 adult-only、视觉小说、诊疗经营或图库奖励。
